## D. 향후 확장 조사 목록 (우선순위 순, 1,000개 목표)

다음 출처를 추가 수집하여 제안서 §4 사업화·기대효과의 [추정] 값을 검증 수치로 교체한다.

### D-1. 소상공인 에너지 통계 (최우선)
1. **소상공인 에너지 비용 실태 통계** — 소진공 실태조사 원문 또는 중소벤처기업부 통계 (에너지 비용 월 평균·업종별 편차)
2. **소상공인 폐업 원인 통계** — 소진공 소상공인 폐업실태조사 (에너지비용 순위)
3. **업종별 에너지 집약도 선행 연구** — 에너지경제연구원(KEEI) 보고서·KCI 논문
4. **업종별 에너지 사용 계절성 분석** — 에너지경제연구원 월보
5. **상업용 건물 에너지 소비 원단위** — 에너지경제연구원 「에너지 소비 원단위 통계」

### D-2. 디지털 서비스 이용 통계
6. **소상공인 디지털 서비스 이용률** — 정보통신정책연구원(KISDI) 통계
7. **소상공인 스마트폰 이용 현황** — 중기부·소진공 통계

### D-3. 에너지 효율화 성과 근거
8. **에너지 효율화 투자 ROI 사례** — 에너지공단 에너지절약마크 사업 결과보고
9. **고효율기기 교체 절감 효과** — 에너지공단 LED·인버터 교체 성과보고
10. **건물 에너지효율등급별 상가 분포** — 국토부 건물에너지통합관리시스템(BEMS)

### D-4. AI·이상치 탐지 선행 연구
11. **AI 기반 에너지 이상치 탐지 선행 연구** — RISS·DBpia 논문 검색 (Isolation Forest 에너지 분야 적용)
12. **건물 에너지 이상치 탐지 논문** — IEEE Xplore·Scopus (Energy Anomaly Detection)

### D-5. 경쟁사 분석
13. **이네르기·에너딕트 등 가격·기능 공개 정보** — 각 사 공식 사이트
14. **한전 파워플래너 이용자 수·기능 상세** — 한전 보도자료·IR

### D-6. 에너지바우처 상세 통계
15. **에너지바우처 미수혜 원인 분석** — 복지부·산업부 협동 보고서
16. **에너지바우처 업종별 수혜 현황** — 산업부 고시 및 결과보고 상세

## E. 데이터 정직성 메모

- 제안서에 사용된 데이터셋 ID: **15101403·15101360·15150932·15129906** — data.go.kr에서 실재 확인 또는 산업부 산하기관 공식 제공 데이터로 등록된 항목
- [추정] 표시된 항목은 파일럿 데이터·추가 조사로 대체 예정
- 날조·가상 ID 없음 — 새 ID를 임의 생성하지 않음
- 출처 확인 중인 항목은 `[확인필요]` 표기
- 비산업부 데이터(환경부 GIR 배출계수·기상청 등)는 "보조" 용도로만, 탈락요건 충족에는 미포함
