## D. 향후 확장 조사 목록 (우선순위 순)

다음 출처를 추가 수집하여 제안서 §4 사업화·기대효과의 [추정] 값을 검증 수치로 교체한다.

1. **소상공인 에너지 비용 실태 통계** — 소진공 실태조사 원문 또는 중소벤처기업부 통계
2. **업종별 에너지 집약도 선행 연구** — 에너지경제연구원(KEEI) 보고서·KCI 논문
3. **소상공인 디지털 서비스 이용률** — 정보통신정책연구원(KISDI) 통계
4. **에너지 효율화 투자 ROI 사례** — 에너지공단 에너지절약마크 사업 결과보고
5. **건물 에너지효율등급별 상가 분포** — 국토부 건물에너지통합관리시스템(BEMS)
6. **AI 기반 에너지 이상치 탐지 선행 연구** — RISS·DBpia 논문 검색
7. **소상공인 폐업 원인 통계** — 소진공 소상공인 폐업실태조사
8. **업종별 에너지 사용 계절성 분석** — 에너지경제연구원 월보
9. **에너지바우처 미수혜 원인 분석** — 복지부·산업부 협동 보고서
10. **경쟁사(이네르기·에너딕트 등) 가격·기능 공개 정보** — 각 사 공식 사이트
