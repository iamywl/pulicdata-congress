# 충전 사각지대 제로 — EV 충전 사각지대 지도 + 대기예측 + 입지추천

## 아이디어 간략 개요

전기차 75.4만 대 시대, 충전기 누적 고장률 17.6% · 수도권 집중(45%) · 고속도로 1기당 494대 대기 문제를 해결하기 위해, 한국에너지공단·한국전력 공공데이터를 결합한 **EV 충전 사각지대 시각화 + AI 대기시간 예측 + 최적 입지 추천** 서비스를 제공한다.
지자체·충전사업자(CPO)에게는 수요-공급 갭 기반 B2G 입지 리포트를, 운전자에게는 실시간 대기예측 기반 '사각지대 없는 충전 경로'를 제공하여 지역 충전 격차를 해소한다.

**핵심 기술·서비스·정보 명칭**: EV-Gap Map(충전 사각지대 지수 지도) / ChargeCast(AI 대기시간 예측 엔진) / SiteScout(수요-공급 갭 기반 입지 최적화 리포트)

## 1. 아이디어 기획 핵심내용 (구체성, 우수성)

### 1.1 서비스 개요 및 핵심 기능

**충전 사각지대 제로**는 세 개의 핵심 기능 모듈로 구성된다.

**① EV-Gap Map — 충전 사각지대 지수 지도**
- 읍·면·동 단위로 "충전 사각지대 지수(CSI: Charging Shortage Index)"를 산출하여 지도 위에 시각화
- CSI = (EV 등록 밀도 × 주행거리 패턴) ÷ (가용 충전기 수 × (1 - 고장률)) 로 정의[추정: 세부 가중치는 데이터 검보정 필요]
- 농어촌·기축아파트·산업단지 주변 등 충전 소외지역을 색상 등고선(등고선형 히트맵)으로 식별
- 출처 데이터: 에너지공단 지역별 급속충전기 현황 + 한전 지역별 충전소 현황[^1][^2]

**② ChargeCast — AI 대기시간 예측 엔진**
- 한전 '전기차 시간대별 충전부하' 데이터를 기반으로, 요일·시간대·기온·특이일(명절·연휴·스포츠경기) 변수를 결합한 **XGBoost + 경량 LSTM 앙상블** 모델로 충전소별 평균 대기시간을 30분 전 예측[^3]
- 모델 입력: 과거 시간대별 충전부하(전력 kWh) → 대기 차량 수 추정 → 대기시간(분) 변환
- 예측 정확도 목표: MAE ≤ 5분 (데이터 충분 지역 기준)[추정]
- 운전자 앱에 "지금 출발하면 N분 대기" 형태로 제공

**③ SiteScout — 수요-공급 갭 기반 입지 최적화 리포트**
- CSI + 교통량 + 인근 상권 체류시간 + 계통 여유용량을 결합한 **Mixed-Integer Linear Programming(MILP)** 최적화로 신규 충전소 최우선 입지 후보 20개를 도출
- 지자체·CPO(충전사업자) 대상 B2G 입지 리포트(PDF/API) 자동 생성
- 출처 데이터: 에너지공단 충전기 지역별 현황 + 한전 충전소 운영정보 + 한전 충전부하 패턴[^1][^2][^3][^4]

## 2. 아이디어 구상 및 제안배경 (활용적정성)

### 2.1 문제 현황 — EV 충전 격차의 4가지 구조적 원인

대한민국 전기차 누적 등록 대수는 2025년 말 기준 75.4만 대로 빠르게 증가하고 있다.[^5] 그러나 충전 인프라는 양적·공간적·기능적으로 뒤처져 있다. 문제는 네 가지로 정리된다.

**① 고장 충전기 — 실효 공급의 구조적 손실**
환경부 전기차 충전 서비스 모니터링 결과(2024), 전국 충전기의 누적 고장률은 17.6%로, 약 5.7대 중 1대가 고장 상태다.[^6] 평균 수리 기간은 3일 이상이며[^6], 이는 운전자가 현장 도착 후 충전 불가 상황에 놓이는 '충전 허탈(Charge Wasted Trip)'을 만든다. 기존 서비스들은 충전기 위치는 안내하지만 고장 이력 기반 신뢰도 지표를 제공하지 않는다.

**② 지역 편중 — 수도권 45% 집중**
에너지공단 급속충전기 현황(2024) 기준, 서울·경기·인천 수도권에 전국 급속충전기의 약 45%가 집중돼 있다.[^7][추정: 공단 데이터 읍면동 집계 기준] 반면 충남·전남·경북 등 농어촌 지역의 EV 보급은 최근 수년간 빠르게 증가했다. 기존 아파트 단지(기축 아파트)와 농어촌 공공시설 인근에는 완속충전기조차 없는 '충전 사막(Charging Desert)' 지역이 다수 존재한다.[^8]

**③ 고속도로 과부하 — 1기당 494대**
한국도로공사 발표(2024) 기준, 고속도로 휴게소의 급속충전기 1기당 등록 EV는 494대다.[^9] 명절·연휴 기간에는 수십 분 ~ 수 시간 대기가 발생하며, 이는 EV 신규 구매 거부감(충전 불안, Range & Queue Anxiety)을 증폭시킨다. 현재 어떤 서비스도 이를 30분 전 예측하여 알려주지 않는다.

**④ 데이터 분절 — 사업자별 폐쇄 생태계**
국내 충전사업자(CPO)는 20개 이상이며 각자 폐쇄적 앱을 운영한다. 결제 시스템도 분절되어 있어 통합 대기·가용성 정보를 수집·분석할 공공 창구가 사실상 없다. 산업통상자원부·에너지공단·한국전력이 개방한 공공데이터가 이 공백을 메울 수 있는 유일한 중립적 기반이다.

**핵심 차별화 3가지**

1. **사각지대 가시화**: 기존 서비스는 '있는 충전기'를 보여준다. 우리는 '없어야 할 곳에 없는 것'을 지도로 보여준다. CSI 히트맵은 정책 입안자와 운전자 모두에게 없던 시각을 제공한다.

2. **AI 사전 예측**: 현재 어떤 서비스도 충전 대기시간을 출발 30분 전 예측하지 않는다. ChargeCast는 한전 시간대별 충전부하 데이터라는 공공 원천을 AI로 변환하여 '도착 전 대기 예측'이라는 새로운 가치를 만든다.

3. **B2G 공급 최적화**: 기존 서비스는 수요자(운전자) 방향이다. SiteScout는 공급자(지자체·CPO) 방향으로 데이터 기반 입지 결정 근거를 제공한다. 이는 13회 수상작들(식품 통관 / 자연어 분석 / 재생에너지 예측)과 명확히 다른 영역이다.

### ④ 창의성 · 독창성

**공공데이터의 역전 활용**: 기존에 에너지공단·한전 데이터는 '있는 충전기'를 관리하는 데 쓰였다. 본 서비스는 이 데이터로 '없어야 할 곳에 없는 충전기'를 역산하여 사각지대를 정의한다. 음(-)의 정보를 양(+)의 정보로부터 도출하는 독창적 접근이다.

**다중 사용자 동시 최적화**: 운전자(단기 대기 최소화)와 지자체/CPO(장기 공급 최적화)라는 상충하는 목표를 하나의 데이터 파이프라인에서 동시에 해결한다.

**오픈 인프라 기반**: 폐쇄적 CPO 데이터 없이, 산업부 공공데이터만으로 작동하는 중립적 플랫폼이다. 이로써 특정 사업자에 종속되지 않고 공공 서비스 성격을 유지한다.

**그림 2.** ChargeCast AI 예측 파이프라인

- **입력 피처**: 과거 24주 × 시간대별 충전부하(kWh), 요일(원-핫), 계절, 기온, 특이일 여부(명절·연휴 플래그), 전일·전주 동시간대 부하(lag 피처)
- **모델 구조**: XGBoost로 정형 피처(요일·기온·특이일)를, Keras LSTM으로 시계열 패턴(연속 시간대 흐름)을 각각 학습 → 가중 앙상블(검증셋 성능 기반 가중치 조정)
- **출력**: 충전소 단위 30분 후 예상 대기시간(분), 신뢰구간(±N분)
- **재학습**: 매주 신규 한전 데이터로 점진적 재학습(incremental learning)
- **AI 해자(why-not-a-wrapper)**: 이 모델은 한전 시간대별 충전부하라는 산업부 공공 독점 데이터로 훈련된다. 외부 LLM API 호출이 아니라, 도메인 전용 시계열 ML 모델 + 에너지 공공데이터 파이프라인이다. 기반 알고리즘이 바뀌어도 한전 데이터로 훈련된 충전 패턴 지식은 우리 자산으로 남는다.

**SiteScout MILP 최적화 상세**

목적함수: 전국 읍면동 CSI 총합 최소화
제약조건:
- 신규 충전소 예산(지자체별 배정 예산 ≤ 제약)
- 전력망 여유용량(계통 여건)
- 기존 충전소와의 최소 거리(중복 투자 방지)
- 우선순위: CSI 상위 20% 지역에 먼저 배치

결과: 지역별 신규 충전소 최적 입지 20개 후보, 예상 CSI 개선 효과(%), 투자 비용 대비 격차 해소 효율(CSI 감소 per 억원)

## 경영혁신·창업학적 프레임워크

### Kim·Mauborgne 블루오션 전략 — 비경쟁 시장 창출

기존 EV 충전 서비스(EV Where, 티맵, 카카오 네비)는 모두 **운전자 → 현재 가용 충전기 탐색**이라는 동일한 레드오션에서 경쟁한다. '충전 사각지대 제로'는 세 가지 비경쟁 가치를 동시에 창출한다:

- **없애기**: 폐쇄 CPO 데이터 의존(중립 공공 데이터로 대체)
- **줄이기**: 운전자의 현장 허탈 경험(예측으로 사전 회피)
- **늘리기**: 정책기관·CPO 대상 B2G 입지 서비스(기존 서비스엔 없음)
- **새로 만들기**: 충전 사각지대 지수(CSI)라는 새로운 인프라 평가 지표

**그림 3.** 전략 캔버스 (블루오션 — 경쟁 요소별 가치곡선)

| 경쟁 요소 | 기존 서비스 평균 | 충전 사각지대 제로 |
|:---|:---:|:---:|
| 위치 안내 정확도 | 높음 | 높음 |
| 실시간 가용 현황 | 중간 | 높음 |
| 대기시간 예측 | 없음 | **높음 (핵심 블루)** |
| 사각지대 지도 | 없음 | **높음 (핵심 블루)** |
| B2G 입지 추천 | 없음 | **높음 (핵심 블루)** |
| 중립성 (비 CPO) | 낮음 | **높음** |

### Christensen 파괴적 혁신 — 저소비자(under-served) 공략

Christensen의 파괴적 혁신 이론에 따르면, 기존 지배적 기업이 도외시한 **저소비자(기존 서비스로 충족 안 된 사용자)**를 먼저 공략하는 것이 성공 경로다. 농어촌·기축아파트 거주 EV 운전자와 충전 인프라 투자 근거가 없어 사업 착수를 못 하는 중소 CPO·지자체가 바로 이 저소비자다. 공공데이터 기반 저비용 서비스로 이들을 먼저 확보한 뒤, 데이터 네트워크 효과로 주류 시장(수도권 운전자)으로 확장한다.

| E10 | IP 없음 | CSI 산출 방법론 특허 출원[추정: 가능성] | 방어적 IP 자산 |

### 차별화 기술의 구매동인 논증

**① 구매동인 가설**

- **운전자(B2C)**: "충전 허탈 없는 이동"은 Must-Have. 현장 도착 후 고장·대기 경험은 EV 구매 후회의 주요 원인[^11]. 대기예측은 이 Must-Have Job을 해결한다.
- **지자체·CPO(B2G·B2B)**: 입지 결정 근거는 Must-Have. 현재 충전소 입지를 데이터 없이 직관으로 결정하는 지자체가 다수이며, 보조금 지원 실패 감사 리스크가 있다. SiteScout는 이 Must-Have에 대응한다.

**② 크기 정량화**

- 운전자: 충전 허탈 1회 = 예상치 못한 대기 30~60분 + 목적지 지연. 이를 회피하는 가치는 운전자 기회비용 기준 1회당 15,000~30,000원[추정]. 월 2회 충전 허탈 회피 시 연간 36~72만원 가치.
- CPO: 충전소 1기 설치 비용 약 1,000만원(급속 기준)[^12]. 입지 오선택 시 가동률 30% 미만 → 수익 회수 불가. SiteScout로 가동률 50% 이상 입지 선택 시 회수기간 2년 단축[추정].

**③ 외부 근거**

- 고장률 17.6% 공식 통계[^6], 고속도로 1기당 494대[^9] — 허탈·대기 문제 실재 입증.
- 유사 서비스 사례: 미국 PlugShare의 실시간 가용 정보 서비스는 EV 운전자 앱 중 최고 평점(4.7/5.0, 구글플레이)[추정: 유사 서비스 사례로 참고], 가용성 정보의 구매동인 강도 입증.

**④ 반증·대안 위협 직시**

- 위협 1: 티맵·카카오 등 대형 플랫폼이 대기예측을 추가할 수 있다. 대응: CSI 데이터 자산과 정책기관(지자체·CPO) 계약 관계는 대형 플랫폼이 빠르게 복제하기 어렵다. B2G 영역은 대형 플랫폼의 수익 우선순위에서 낮다.
- 위협 2: 무료 대안(환경부 앱 등) 존재. 대응: 환경부 앱은 현재 상태 표시에 머물며 예측·사각지대·입지 추천이 없다.

## 4. 아이디어의 사업화방안 및 기대효과 (사업성, 실현가능성)

### 4.1 시장성 — TAM / SAM / SOM

| 구분 | 규모 | 근거 |
|:---|:---|:---|
| TAM (전체 시장) | 약 3조 2,000억원 | 국내 EV 충전 인프라 시장 2030년 전망[^13] |
| SAM (유효 시장) | 약 4,800억원 | CPO·지자체 입지 컨설팅 + 데이터 서비스 시장 [추정] |
| SOM (초기 목표) | 약 24억원 | 지자체 30개 × 연 800만원 + CPO 10개 × 연 2,400만원 (3년차 목표)[추정] |

**Why Now**: 2025년 말 EV 75.4만 대, 2030년 420만 대 목표[^10]. 지금이 충전 인프라 격차 가시화 + 공공 투자 정책 시행 시점. 데이터 기반 입지 서비스 수요가 폭발적으로 증가할 '골든윈도우'다.

### 4.2 고객확보 (Go-to-Market)

**타깃 고객 세분화 (ICP)**

- ICP-1 (B2G): 농어촌·기초 지자체 교통·에너지 담당 부서, 예산 5천만원~5억원 충전 인프라 사업 보유
- ICP-2 (B2B CPO): 신규 입지 개발 중인 중소형 CPO 5~50기 운영 규모
- ICP-3 (B2C): 농어촌·교외 거주 EV 운전자, 장거리 주행 빈번 사용자

**초기 트랙션 확보 계획 (첫 100 파일럿 지자체 담당자)**

1. 충전 사각지대 지수(CSI) 상위 10개 기초자치단체에 무료 파일럿 리포트 발송 (콜드 아웃리치)
2. 산업통상자원부·에너지공단 주최 EV 충전 정책 포럼 발표 참여 → 레퍼런스 구축
3. 지자체 채택 후 수상/언론 보도 → 추가 지자체 확산 (레퍼런스 세일즈)

**퍼널 (B2G)**
인지(포럼·보도) → 무료 샘플 리포트(트라이얼) → 유료 정기 구독(분기 리포트) → 추가 분석 의뢰(업셀)

**예상 CAC (B2G)**: 지자체당 약 50만원[추정] (영업·포럼 참여비 배분)
**리텐션 가설**: 연간 계약(분기 리포트 4회), 예산 사이클 기반 갱신 → 이탈률 낮음
