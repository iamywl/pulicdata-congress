# 통상 레이더 — 관세·통상 리스크 조기경보 모니터링

> KOTRA 공공 통상·무역 뉴스 데이터를 LLM 기반 이슈 분류·영향도 스코어링 엔진으로 가공하여, 기업 사용자의 **수출 품목·대상국**에 직접 영향을 미치는 관세·통상정책 변화만 선별 추출하고, 조기경보(푸시 알림)·리포트로 능동 전달하는 B2B SaaS 서비스.

**핵심 기술·서비스·정보 명칭**

- **KOTRA 미국 글로벌 이슈 모니터링 API** (데이터셋 15134045) — 실시간 미국발 통상이슈
- **KOTRA 단신속보뉴스 API** (데이터셋 15122662) — 국가별 단기 경제·정책 속보
- **KOTRA 해외시장뉴스 API** (데이터셋 15034831) — 시장·상품·통상 심층 분석 기사
- **LLM 이슈 분류·영향도 스코어링 엔진** — 품목(HS코드) × 국가 × 이슈 유형 3-축 매핑
- **품목-국가 맞춤 조기경보 시스템** — 임계 리스크 발생 시 즉각 알림 발송

## 1. 아이디어 기획 핵심내용 (구체성, 우수성)

### 1.1 무엇을 만드는가

**통상 레이더(Trade Radar)**는 KOTRA가 공개하는 통상·무역 뉴스 데이터를 실시간으로 수집하고, LLM 기반 엔진으로 분석하여 **"내 품목·내 수출국에 영향을 주는 이슈만"** 골라내어 조기경보를 발송하는 서비스다.

기존의 통상 정보는 일반 뉴스 형태로 배포된다. 실무자가 이를 읽어 본인 업무와의 관련성을 판단해야 하는데, KOTRA 단신속보뉴스만 해도 연간 약 8,000건 이상 발행된다. 이 중 한 기업이 실제로 관심을 가져야 할 이슈는 수십 건에 불과하다. 이 신호-잡음비 문제를 해결하는 것이 본 서비스의 핵심이다.

### 1.2 핵심 기능 5가지

| 번호 | 기능명 | 내용 |
|:---:|:---|:---|
| F-1 | 품목·국가 프로필 등록 | 사용자가 수출 HS코드(4~10단위)와 대상국을 등록하면 개인화 필터 생성 |
| F-2 | 실시간 이슈 수집·분류 | KOTRA 3개 API를 6시간 주기 수집, LLM으로 이슈 유형(관세/인증/제재/규제/환율)·영향도 점수 부여 |
| F-3 | 조기경보 발송 | 영향도 점수 임계값(사용자 설정) 초과 시 이메일·슬랙·카카오톡 즉시 발송 |
| F-4 | 맞춤 리포트 | 주간·월간 요약 리포트, 이슈 추이 타임라인, 리스크 히트맵(품목 × 국가) |
| F-5 | 규제 변경 추적 | 동일 품목·국가에 대한 이슈를 시계열로 연결해 규제 변화 흐름 시각화 |

### 1.3 AI 활용 방식 (구체)

본 서비스의 AI 처리는 단순 API 래퍼가 아니라, 도메인 특화 파이프라인 4개 레이어로 구성된다.

**레이어 1 — 이슈 유형 분류 (LLM + 프롬프트 온톨로지)**
KOTRA 뉴스 텍스트를 입력받아 관세/비관세장벽/제재·수출통제/인증·규격/환경규제/바이어 신용의 6개 유형으로 분류한다. 이 유형 체계(온톨로지)는 산업통상자원부 통상 백서, WTO 분류 기준을 참조해 사전 설계하며, 서비스 고유 자산이 된다.

**레이어 2 — 영향도 스코어링**
분류된 이슈에 대해 (a) 규제 변화 강도, (b) 시행 임박성, (c) 해당 산업의 대미·대EU 의존도, (d) 유사 이슈의 과거 영향 사례를 결합하여 0~100점 영향도 점수를 산출한다. 스코어링 로직은 도메인 전문 룰과 LLM 추론을 결합한다.

**레이어 3 — 품목-HS코드 매핑**
이슈 본문에서 언급된 제품·산업·재료 표현을 HS 코드(6단위)로 자동 매핑한다. 관세청 HS 코드 체계를 참조 데이터로 활용하고, LLM으로 자연어 → HS 코드 변환을 수행한 뒤 사용자의 등록 HS 코드와 비교하여 관련성을 판단한다.

**레이어 4 — 개인화 필터 및 알림 트리거**
사용자별 HS 코드 × 대상국 프로필에 레이어 1-3 결과를 매핑하여, 해당 사용자에게 유관한 이슈만 통과시킨다. 영향도 점수가 사용자 설정 임계값(기본값 70점)을 초과하면 즉시 경보를 발송한다.

## 2. 아이디어 구상 및 제안배경 (활용적정성)

### 2.1 관련 현황 및 문제점

2025~2026년 미국을 중심으로 한 통상환경 급변은 한국 수출 기업, 특히 중소·중견 기업에 심각한 리스크 요인이 되고 있다.

**배경 지표**

- 한국 수출에서 대미 수출 비중은 2025년 기준 약 18.7%로 1위권 [^1]. 미국발 관세 정책 하나가 대형 충격을 준다.
- 2025년 미국은 철강·알루미늄 25% 관세 재부과, 반도체·배터리 섹터별 관세를 통보 없이 단기 도입하는 패턴을 반복했다.
- 중소기업 수출업체 중 전담 통상팀 보유 비율은 5% 미만으로 추정되며[추정], 대다수가 외부 정보를 일반 뉴스 검색에 의존한다.
- KOTRA 단신속보뉴스의 연간 건수는 8,000건 이상 [^2]. 이를 매일 검토하는 중소기업 실무자는 사실상 없다.

### 2.2 활용적정성 4요소

**① 활용분야**
수출 중소·중견 기업의 통상리스크 모니터링. 산업군별로는 철강·금속(관세 직격), 반도체·전자(수출통제), 자동차·부품(원산지), 화학·바이오(규제·인증), 섬유·의류(반덤핑) 등 거의 전 제조업 수출 품목에 적용된다.

**② 활용빈도**
SaaS 구독 모델 — 서비스는 6시간 주기로 데이터를 수집·처리하고, 사용자는 대시보드를 일상적으로 확인한다. 경보 발생 시에는 즉각(실시간) 알림이 발송되므로, 경보일에는 하루 수회 이상 접속이 발생한다.

**③ 활용범위**
직접 수출 기업뿐 아니라, 수출 기업을 고객으로 두는 무역 컨설팅사·세관 전문가·중소기업진흥공단·KOTRA 지역 무역관 컨설턴트도 2차 활용 대상이다. 국내 수출 중소기업 약 96,000개사가 잠재 대상이다 [^3].

**④ 중요성**
통상정책 변화 대응에 골든타임이 있다. 미국 무역확장법 232조 관세처럼, 공식 발표 후 수 주 내에 수입 계약·재고·가격 조정을 해야 손실을 최소화한다. 이 골든타임 안에 정보를 포착·해석하지 못하는 기업은 관세 인상분 전액을 비용으로 부담하게 된다. KOTRA 공개 데이터에는 이미 이 정보가 있으나, '내 품목에 해당하는가'를 자동으로 필터링하는 도구가 없다 — 이것이 본 서비스가 채우는 공백이다.

## 3. 아이디어 세부내용

### ① 활용 산업부 공공데이터 (탈락요건 필수 항목)

본 서비스는 아래 3개 산업통상자원부 산하기관(KOTRA) 공공데이터를 핵심 데이터 소스로 활용한다.

| 번호 | 데이터셋명 | 기관 | 데이터셋 ID | data.go.kr URL |
|:---:|:---|:---|:---:|:---|
| D-1 | KOTRA 미국 글로벌 이슈 모니터링 | KOTRA (산업통상자원부 산하) | 15134045 | https://www.data.go.kr/data/15134045/openapi.do |
| D-2 | KOTRA 단신속보뉴스 | KOTRA (산업통상자원부 산하) | 15122662 | https://www.data.go.kr/data/15122662/openapi.do |
| D-3 | KOTRA 해외시장뉴스 | KOTRA (산업통상자원부 산하) | 15034831 | https://www.data.go.kr/data/15034831/openapi.do |

**D-1 KOTRA 미국 글로벌 이슈 모니터링**: 미국발 정책·경제·기술 이슈를 전문 무역관이 작성하는 시리즈. 관세·수출통제·제재·공급망 재편 등 통상 핵심 이슈를 가장 신속하게 커버한다. 본 서비스의 1차 경보 소스.

**D-2 KOTRA 단신속보뉴스**: 전 세계 무역관이 국가별 단기 경제·정책 이슈를 속보 형식으로 발행. 연간 8,000건+의 방대한 물량으로, 본 서비스의 LLM 이슈 분류 엔진이 가장 많이 처리하는 피드.

**D-3 KOTRA 해외시장뉴스**: 국가별·산업별 심층 분석 기사. 속보보다 깊이 있는 시장 분석 정보를 제공하며, 영향도 스코어링의 배경 컨텍스트로 활용된다.

**활용 방법**: 3개 API를 6시간 주기로 자동 호출하여 신규 기사를 수집하고, 파이프라인에 투입한다. API는 data.go.kr 오픈 API 정책(회원가입·활용신청 무료)에 따라 활용한다.

### ④ 창의성·독창성

기존 서비스와의 핵심 차이는 **"뉴스 소비"에서 "리스크 경보"로의 패러다임 전환**이다. KOTRA가 방대한 양질의 통상 정보를 공개하고 있음에도, 이를 기업이 소화할 수 있는 형태로 개인화하는 인터페이스가 없었다. 본 서비스는 공공데이터를 원료로, LLM 파이프라인을 공장으로, 기업 맞춤 경보를 완제품으로 전달하는 새로운 가치사슬을 만든다.

특히 HS코드 기반 개인화는 기존 어떤 서비스도 시도하지 않은 접근이다. 동일한 미국 관세 뉴스라도 철강 기업과 반도체 기업의 관련성은 전혀 다르다. 이 품목-이슈 매핑을 자동화한 것이 본 서비스의 창의적 핵심이다.

13회 수상작과의 차별성:
- **식품 통관도우미**: 수입 통관에 집중, 수출 통상리스크 아님
- **shannon(자연어 데이터분석)**: 데이터 분석 도구, 통상 도메인 아님
- **MLP-XGB 기상예측 오차보정**: 재생에너지·전력 도메인

본 서비스는 '수출 중소기업의 통상리스크 조기경보'라는 독자적 영역을 다룬다.

**AI 해자 논증 — 래퍼가 아닌 이유**

본 서비스가 단순 LLM API 래퍼가 아닌 근거:

1. **독자 온톨로지**: WTO 통상 분류·관세청 HS 코드 체계를 결합한 이슈 분류 체계를 사전 설계한다. 이 체계는 본 서비스 고유 자산이며, 어떤 LLM에도 프롬프트로 재현되지 않는다.
2. **피드백 루프 데이터**: 사용자가 이슈 관련성을 평가(유관/무관)하면, 이 평가 데이터가 HS코드 매핑 정확도 개선에 사용된다. 이 사용자 생성 학습 데이터는 경쟁자가 복제 불가능한 자산이다.
3. **RAG 기반 과거 이슈 검색**: 누적된 이슈 DB와 벡터 인덱스를 RAG로 활용하여 현재 이슈의 역사적 맥락을 제공한다. 이 이슈 DB는 시간이 지날수록 깊어지는 데이터 해자다.
4. **LLM 교체 가능 구조**: 분류·스코어링 인터페이스를 추상화하여, GPT-4o-mini → Llama → Claude 등으로 교체해도 파이프라인이 그대로 작동하도록 설계한다. 서비스 가치는 모델이 아니라 데이터·온톨로지·워크플로에 있다.

**서비스 방법**

서비스는 웹 SaaS로 제공된다. 신규 사용자는 회원가입 → 수출 품목(HS코드 4~10자리) 등록 → 대상국 선택 → 알림 채널 설정의 4단계(3분 이내)를 완료하면 즉시 경보 수신을 시작한다. 기업 규모에 따라 싱글 유저·팀 계정을 선택할 수 있다.

### 4.3 고객확보 (Go-to-Market)

**ICP (이상 고객 프로필)**
- 수출 비중 30%+ 중소 제조기업
- 미국·EU·중국 동시 수출 (복수 통상리스크 노출)
- 전담 통상팀 없음 (담당자 1인)
- 연매출 50억~500억 원 구간

**초기 100 사용자 확보 (0→1 단계)**
1. **KOTRA 파트너십**: KOTRA 지역본부 중소기업 세미나·상담회 참여 → 무료 플랜 바이럴 유도. KOTRA는 산업부 산하기관으로 공공 협력 가능성이 높다.
2. **중소기업수출지원센터·중진공 연계**: 수출 초보 기업 대상 교육 프로그램에 무료 체험 연계.
3. **산업별 협회 채널**: 한국철강협회·반도체산업협회 등 협회 뉴스레터에 파트너 광고.
4. **콘텐츠 마케팅**: 미국 관세 이슈가 터질 때마다 무료 분석 리포트 발행 → 리드 수집.

**획득 퍼널**: 인지(세미나·뉴스레터) → 무료 체험(스타터) → 전환(프로/팀) → 유지(월간 리포트·알림 가치 체감)

**리텐션 가설**: 알림 가치는 관세 이슈가 발생하는 시점에 극대화된다. 사용자가 알림을 통해 실제 리스크를 회피한 경험을 하면 이탈률이 급감한다. 온보딩 단계에서 "첫 관련 이슈 알림 수신 체험"을 빠르게 제공하는 것이 리텐션의 핵심이다.

### 4.4 경영혁신·창업학적 프레임워크

**Jobs To Be Done (JTBD) 분석**

통상 레이더는 JTBD 프레임워크로 정확히 설명된다. 수출 중소기업 담당자의 핵심 JTBD는 "우리 제품에 영향을 주는 통상 변화를 남들보다 빨리 파악하여 경영진에게 보고하고 의사결정을 도와주고 싶다"이다. 기존 수단(KOTRA 사이트 방문, 구글 검색)은 이 일을 불완전하게 수행한다 — 이 채용 공백(hiring gap)이 본 서비스의 시장 기회다.

**정보 비대칭 제거 — 파괴적 혁신 (Christensen)**

통상 인텔리전스는 전통적으로 대기업만 접근 가능한 고가 컨설팅 서비스다. 본 서비스는 공공데이터 + AI로 이 정보 비대칭을 해소하여, 중소기업이 대기업 수준의 통상리스크 모니터링 역량을 저비용으로 갖출 수 있게 한다. 이는 Christensen의 파괴적 혁신 — 기존 과소서비스(underserved) 시장을 신기술로 공략하는 전형적 패턴에 해당한다.

**Why Now**

미국발 관세 정책의 변동성 증가(2025~2026 기점), LLM의 텍스트 분류 능력 급향상, 그리고 KOTRA의 공공데이터 개방 확대가 동시에 발생하는 현 시점이 이 서비스를 출시할 최적기다. 2년 전에는 LLM 비용이 너무 높았고, 2년 후에는 대형 경쟁사가 유사 서비스를 출시할 가능성이 높다.

### 4.5 차별화 기술의 구매동인 논증

**① 구매동인 가설**

핵심 구매동인: **"관련 없는 정보를 걸러주는 개인화 필터 + 임박한 리스크의 능동 알림"**. 이는 must-have 수준이다. 현재 수출 담당자는 연간 8,000+ 건의 KOTRA 뉴스를 검토할 시간이 없어 사실상 정보를 포기한 상태다. 본 서비스 없이는 리스크를 놓치는 것이 기본값이다.

**② 크기 정량화**

- 관세 이슈 대응 골든타임 실패 시: 관세 인상분 (예: 10%p) × 해당 품목 미국 수출액 전액 부담 → 연매출 50억 기업 대미 수출 비중 30% 기준 약 1.5억 원 추가비용 [추정]
- 서비스 연 구독료(프로 기준): 약 108만 원
- ROI 비율: 약 139배 [추정] (1건의 리스크 회피만으로도 구독료 100년분 상환)
- 뉴스 검토 시간 절감: 월 10시간 × 인건비(중소기업 과장급 시간당 3만 원) = 월 30만 원 절감 [추정] → 연 360만 원 vs 연 구독료 108만 원

**③ 외부 근거**

- 수출 중소기업 1년 생존율 49.2% [^3] — 통상리스크 대응 실패가 주요 원인 중 하나. 조기경보 서비스는 생존율 개선에 직결.
- FTA 활용률 격차(중소 61.5% vs 대기업 85.1%, [^4]) — 정보 비대칭이 실제로 존재함을 증명. 통상 레이더는 이 격차를 메우는 수단.

**④ 반증·대안 위협**

- 위협 1: 기업이 "KOTRA 사이트 직접 보면 된다"고 판단할 수 있음 → 대응: 연 8,000건 검토 vs 관련 건만 수신의 실무 비교 데모로 반증.
- 위협 2: 대형 무역 플랫폼이 유사 기능 추가 → 대응: 도메인 특화 온톨로지·피드백 데이터는 단기 복제 불가. 먼저 쌓는 데이터 해자가 방어선.
- 위협 3: 가격 민감도 (중소기업 SaaS 구독 거부감) → 대응: 프리티어로 진입 마찰 제거, 첫 리스크 회피 경험으로 전환.
