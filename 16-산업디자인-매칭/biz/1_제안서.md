# 디자인 브릿지 — 중소기업–산업디자인전문회사 AI 매칭

## 아이디어 간략 개요

중소기업이 업종·제품 카테고리·예산·디자인 니즈를 입력하면, AI가 전국 18,014개 산업디자인전문회사 데이터와 굿디자인(GD) 수상 트렌드를 결합해 최적 디자인사를 추천·연결하는 플랫폼이다.
탐색 시간을 80% 이상 단축하고, 중소기업의 첫 디자인 협업 성공률을 높이는 것이 목표다.

## 핵심 기술·서비스·정보 명칭

- **AI 니즈-전문분야 매칭 엔진** (벡터 임베딩 + 협업 필터링 하이브리드)
- **굿디자인(GD) 트렌드 연계 추천** (한국디자인진흥원 공모전 목록 데이터 활용)
- **산업디자인전문회사 데이터베이스** (data.go.kr 기반, 18,014건)

## 1. 아이디어 기획 핵심내용 (구체성, 우수성)

### 1-1. 무엇을 만드는가

"디자인 브릿지"는 **중소기업 ↔ 산업디자인전문회사를 AI로 연결하는 매칭 플랫폼**이다. 다음 세 기능을 하나의 웹 서비스로 통합한다.

| 기능 | 설명 |
|:---|:---|
| **니즈 기반 매칭** | 기업의 업종·제품·디자인 목적·예산·납기를 자연어로 입력 → AI가 적합 디자인사 Top 5~10 추천 |
| **GD 트렌드 연계** | 굿디자인 수상 데이터와 연계, 업종별 최근 수상 실적이 있는 디자인사를 가중 추천 |
| **투명한 포트폴리오 탐색** | 디자인사의 전문분야·지역·규모·수상 이력을 카드 UI로 비교·탐색 |

**그림 1.** 디자인 브릿지 AI 매칭 파이프라인

**① 임베딩 레이어**: 한국어 특화 SBERT 계열 모델(예: KoSimCSE)로 디자인사의 전문분야 텍스트와 기업 니즈 텍스트를 각각 벡터화한다. 이 벡터 DB는 자체 구축되며, 기반 LLM이 교체되더라도 벡터 인덱스와 도메인 온톨로지는 플랫폼 고유 자산으로 남는다.

**② 협업 필터링**: 유사 업종 기업이 선택한 디자인사 이력(익명 집계)을 기반으로 콜드스타트 이후 추천 품질을 높인다. 사용 데이터가 쌓일수록 정확도가 향상되는 데이터 네트워크 효과가 발생한다.

**③ GD 트렌드 가중치**: 한국디자인진흥원 공모전 목록(데이터셋 15121448)에서 최근 3년 수상 실적을 파싱해, 동일 업종 수상 경험이 있는 디자인사에 가중치를 부여한다. 단순 디렉터리 검색과 달리 "트렌드에 맞는 디자인사"를 기준으로 필터링하는 차별점이다.

**④ 추천 이유 설명(Explainable AI)**: 추천 결과에 "귀사 업종과 유사한 제조 중소기업 7곳이 협업했고, 최근 2년 내 GD 수상 실적 2건 보유" 같은 구체적 근거를 표시해 신뢰도를 높인다.

### 1-3. 우수성 요약

- **산업부 공공데이터 직접 활용**: 탈락요건 충족 — 산업디자인전문회사 DB와 GD 공모전 목록 모두 산업통상자원부 산하 한국디자인진흥원(KIDP) 소관 데이터
- **AI 방식 구체화**: 단순 LLM API 래퍼가 아니라, 자체 도메인 벡터 DB + 매칭 이력 협업 필터링 + GD 트렌드 연계 파이프라인으로 독자 해자 확보
- **기존 서비스 공백 정확히 타격**: 디자인사 검색·매칭에 특화된 공공 서비스가 사실상 부재

## 2. 아이디어 구상 및 제안배경 (활용적정성)

### 2-1. 현황 및 문제점

**전국 산업디자인전문회사는 18,014개(2024년 기준)**[^1]에 달하지만, 중소기업이 자사 니즈에 맞는 디자인사를 찾는 체계적인 경로는 없다. 현재 주요 탐색 방법은 ① 지인 소개·구전, ② 포털 검색("디자인 회사 추천"), ③ 한국디자인진흥원 디자인DB 단순 디렉터리 열람 등이다. 이는 정보 비대칭과 탐색 비용을 구조적으로 높인다.

| 문제 | 수치·근거 |
|:---|:---|
| 탐색 대상 규모 | 전국 18,014개 산업디자인전문회사[^1] |
| 중소기업 디자인 투자 비율 | 전체 사업체의 86% 이상이 디자인 전담인력 없음[^2] [추정: 중소기업 기준 추계] |
| 평균 디자인사 선정 소요 시간 | 약 3~8주(구전 의존 탐색 포함)[추정] |
| 굿디자인 수상 현황 | 연간 약 600~800개 제품 선정[^3] |

**[추정]** 중소기업 디자인 탐색 소요 시간 3~8주는 업계 인터뷰 기반 추정값이며 공식 통계 미확인. 향후 고객 인터뷰로 검증 예정.

### 2-2. 활용분야·활용빈도·활용범위·중요성 4요소

**① 활용분야**
- **주 활용 분야**: 중소기업 제품디자인 외주 발굴 — 제조업 중소기업이 신제품 개발·리뉴얼 시 디자인 협력사를 찾는 과정
- **부 활용 분야**: 디자인사의 신규 수주처 발굴(양방향 매칭), 지역 디자인 생태계 활성화 정책 지원

**② 활용빈도**
- 중소기업 관점: 연 1~3회 수준(신제품 개발·모델 변경 주기)
- 디자인사 관점: 상시(포트폴리오 노출·수주 확장)
- 플랫폼 관점: 기업 회원 가입 → 매칭 요청 → 추천 열람 → 연락 → 재방문 이력 → 협업 필터 업데이트 (데이터 루프)

**③ 활용범위**
- 지리적 범위: 전국(서울·수도권 집중에서 지역 디자인사로 분산 유도 가능)
- 산업 범위: 제조업(가전·소비재·산업기계·식품용기 등) + 서비스업(브랜딩·패키지) 전 업종
- 수혜 규모: 잠재 사용 중소기업 약 10만~20만 개사[^4][추정], 등록 디자인사 18,014개

**④ 중요성**
한국 중소기업의 디자인 경쟁력 강화는 수출 제품의 글로벌 경쟁력과 직결된다. 산업통상자원부가 굿디자인 제도를 통해 디자인 진흥 정책을 추진하는 핵심 취지가 중소기업 제품 경쟁력 향상이다[^5]. 그러나 디자인 협력사 발굴 단계에서의 정보 격차가 이 정책 효과를 약화시키고 있다. 공공데이터(18,014개 디자인사 정보 + GD 공모전 데이터)를 결합한 AI 매칭으로 이 병목을 해소하면, 산업부 디자인 정책의 말단 전달력을 실질적으로 높일 수 있다.

### 2-3. 경영혁신·창업학적 프레임워크

**JTBD(Jobs To Be Done) + 블루오션 전략**을 적용한다.

- **JTBD 관점**: 중소기업이 디자인사를 찾는 진짜 "Job"은 "디자인사 목록을 보는 것"이 아니라 "빠르게 신뢰할 만한 협력사를 찾아 제품 경쟁력을 높이는 것"이다. 현재의 단순 디렉터리 서비스는 이 Job을 완전히 해결하지 못한다. 디자인 브릿지는 이 Job을 정확히 타격한다.

- **블루오션 관점**: 기존 디자인 플랫폼(크몽·오투잡·탤런트뱅크)은 프리랜서 디자이너 매칭에 집중하고, 산업디자인전문회사(법인 디자인사) 매칭 영역은 사실상 경쟁 공백이다. 공공데이터 기반 B2B 산업디자인사 매칭은 새로운 가치 곡선을 그린다.

- **Why Now**: 산업통상자원부의 "산업디자인전문회사" 공공데이터(18,014건)가 2024년 data.go.kr에 공개되어 있고, GD 공모전 목록 데이터도 API로 접근 가능하다. 이 데이터를 AI로 연결하는 타이밍이 지금이다.

### ④ 창의성·독창성

1. **공공데이터 활용 매칭**: 민간 플랫폼이 수집하기 어려운 산업부 산하 18,014개 디자인전문회사 전수 데이터를 공공데이터 경로로 정식 확보해 매칭 DB로 전환한다. 이는 민간 사업자가 복제하기 어려운 구조적 진입 장벽이다.

2. **GD 트렌드 연계 추천**: 굿디자인 수상 데이터를 추천 알고리즘 가중치로 결합하는 방식은 기존 매칭 플랫폼에서 찾아볼 수 없는 독창적 접근이다.

3. **Explainable AI**: "왜 이 디자인사를 추천했는가"를 고객 언어로 설명하는 방식은 B2B 디자인 매칭에서 신뢰 결정에 필수적이다.

4. **정책-시장 이중 효과**: 산업부 디자인 진흥 정책의 말단 실행력 강화(디자인사 DB 활성화)와 중소기업 디자인 역량 향상이 동시에 발생한다.

**AI 해자 논증 (API 래퍼가 아닌 이유)**

- **독자 자산**: 벡터 DB(18,014개 디자인사 임베딩 인덱스)와 도메인 온톨로지(KSIC ↔ 디자인분야 매핑), 협업 필터 누적 이력은 플랫폼 고유 자산으로, 기반 LLM을 GPT-4에서 다른 모델로 교체해도 이 자산은 그대로 남는다.
- **버티컬 워크플로**: 공공데이터 수집 → 정제 → 임베딩 → 협업 필터 → GD 가중치 결합 → Explainable 출력까지의 전 파이프라인이 B2B 디자인 매칭에 특화된 고유 워크플로다. 단발 생성(ChatGPT 프롬프트)으로 대체 불가.
- **데이터 네트워크 효과**: 매칭 이력이 쌓일수록 협업 필터 정확도가 높아져 후발 경쟁자의 복제가 어려워진다.
- **모델 교체가능성 전제**: 기반 LLM은 상품화된다. 우리 레이어는 "한국 산업디자인 도메인 특화 벡터 인덱스 + 매칭 이력 + GD 가중치 규칙"이며, 모델이 바뀌어도 이 레이어는 경쟁 우위를 유지한다.

### 차별화 기술의 구매동인 논증

**① 구매동인 가설 명시**

중소기업이 디자인사를 찾지 못해 제품 출시가 지연되거나 부적합 디자인사와의 협업으로 비용·시간을 낭비하는 것은 **must-have 문제**다. "있으면 좋음(nice-to-have)"이 아니라, 해결 못 하면 신제품 개발 자체가 막히는 병목이다.

**② 크기 정량화**

- 탐색 소요 시간: 현재 약 3~8주 [추정] → 플랫폼 이용 시 1~3일로 단축 (약 80% 단축) [추정]
- 부적합 협업 비용 절감: 초기 디자인 발주 실패 시 재발주 비용 평균 500~2,000만원 [추정, 업계 인터뷰 기반] → 적합사 매칭으로 재발주율 감소
- 전환 비용: 무료 플랜 → 기존 구전 방식 대비 추가 비용 없음, 전환 마찰 극히 낮음

[추정] 위 수치는 현 단계 추정값. 론칭 후 고객 인터뷰·A/B 테스트로 검증 예정.

**③ 외부 근거로 뒷받침**

- 중소기업 디자인 전담인력 부재 비율 86%[^2]: 전문 역량이 없으니 외부 협력사 탐색이 어렵고, 매칭 플랫폼의 가치가 더욱 크다.
- 굿디자인 제도 운영 현황 및 디자인 진흥 효과[^5]: 산업부가 디자인 경쟁력 강화를 정책 우선순위로 두는 근거.

**④ 반증·대안 위협 직시**

- **위협 1**: 중소기업이 "그냥 포털 검색으로 찾겠다" — 대응: 18,014개 전수 + GD 연계 + Explainable 추천이 포털 검색 결과보다 신뢰도·관련성이 높음을 초기 사용 경험에서 입증해야 한다.
- **위협 2**: 디자인사가 등록을 안 한다 — 대응: 공공데이터 기반이므로 디자인사의 자발적 등록이 필요 없음. 이미 18,014개사 전체가 DB에 포함되어 있음.
- **위협 3**: 대형 플랫폼(크몽·탈잉)이 기능을 복제한다 — 대응: 공공 디렉터리 데이터와 GD 연계 구조는 민간 플랫폼이 복제하기 어려운 공공 협력 우위. 산업부·KIDP와의 공식 협력 관계 구축이 선점 해자.
