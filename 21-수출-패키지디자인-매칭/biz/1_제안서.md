# 글로벌 패키지 — 현지 선호 기반 수출 패키지디자인 매칭

- KOTRA 국가별 시장정보·해외시장뉴스를 LLM으로 분석해 타깃 국가의 패키지 디자인 선호(색채·문구·소재·인증마크 등)를 자동 추출하고, 한국디자인진흥원 산업디자인전문회사 데이터베이스와 연결해 수출 중소기업에 맞춤 디자인사를 매칭하는 서비스.
- 수출 중소기업의 현지화 패키지 설계 기간을 단축하고, 영세 기업도 국가별 패키지 가이드라인을 즉시 확인할 수 있다.

**핵심 기술·서비스·정보 명칭**
- KOTRA 국가정보 API / KOTRA 해외시장뉴스 API / 한국디자인진흥원 산업디자인전문회사 데이터
- 국가별 선호 속성 추출 LLM 파이프라인 (RAG 기반)
- 패키지 디자인 가이드 자동 생성 엔진
- 수출품목 × 타깃국 × 디자인 역량 3-way 매칭 알고리즘

## 1. 아이디어 기획 핵심내용 (구체성, 우수성)

### 1-1. 무엇을 만드는가

**글로벌 패키지**는 수출 중소기업이 타깃 국가를 입력하면 ① 해당 국가의 패키지 디자인 선호 가이드(색채·레이아웃·문구 금기어·필수 인증마크·소재 규제 등)를 즉시 제공하고, ② 기업 품목·예산·일정에 맞는 산업디자인전문회사를 추천·연결하는 웹 서비스다.

### 1-2. 3단계 핵심 기능

**표 1.** 핵심 기능 요약

| 단계 | 기능 | 데이터 원천 | 출력물 |
|:---|:---|:---|:---|
| 1 | 타깃국 패키지 선호 분석 | KOTRA 국가정보·해외시장뉴스 (RAG) | 국가별 패키지 속성 카드 |
| 2 | 패키지 디자인 가이드 생성 | LLM + 분석 결과 | 색채 팔레트·레이아웃 규칙·금기어·인증마크 |
| 3 | 디자인사 매칭 | 한국디자인진흥원 산업디자인전문회사 DB | 상위 3~5개 추천 + 연락처 |

### 1-3. 구현 기술 개요

- **데이터 수집·정제**: KOTRA 국가정보 API(15034830)에서 264개국 시장 분석 문서, KOTRA 해외시장뉴스 API(15034831)에서 최신 트렌드 기사를 정기 수집·벡터화(임베딩 모델: `text-embedding-3-small` 또는 오픈소스 `bge-m3`).
- **RAG 파이프라인**: 기업이 입력한 품목·국가 쿼리에 대해 벡터DB(ChromaDB)에서 관련 문서 청크를 검색, LLM(GPT-4o 또는 Claude 3.5 Sonnet)에 컨텍스트로 주입해 패키지 속성을 구조화 JSON으로 추출한다.
- **가이드 생성 엔진**: 추출된 속성(색채 선호, 금기 색·문구, 필수 인증, 소재 규제, 폰트 선호)을 HTML/PDF 가이드 카드로 렌더링.
- **매칭 알고리즘**: 한국디자인진흥원 산업디자인전문회사 데이터(18,014건)에서 전문분야 필터(패키지·제품) → 지역 필터 → 규모 필터 → LLM 기반 역량 스코어링 → 상위 N개 추천.
- **독자 자산(API 래퍼가 아닌 이유)**: KOTRA 문서를 그대로 노출하는 것이 아니라, 패키지 디자인 도메인 온톨로지(색채 심리학, 인증마크 체계, 수출입 소재 규제)를 룰베이스로 구축하고 LLM 추론 결과를 교차 검증한다. 사용 피드백(기업별 매칭 성사율, 가이드 만족도)을 재학습 데이터로 축적해 데이터 네트워크 효과를 형성한다.

## 2. 아이디어 구상 및 제안배경 (활용적정성)

### 2-1. 문제 현황

수출 중소기업 95,905개사 중 절반이 수출 1년 내에 중단한다(1년 생존율 49.2%)[^1]. 그 원인 중 하나는 **현지화 부족**이다. 패키지·디자인은 현지 소비자의 구매 결정에서 첫 인상을 결정하는 요소지만, 중소기업은 이를 조사하고 전문사에 연결할 인력·예산이 없다.

구체적인 수치를 보면:
- 수출 중소기업이 해외인증·현지화에 투입하는 비용은 연 약 2.3조 원으로 추정되며[^2], 이 중 패키지 디자인 현지화 비용은 별도 데이터가 없으나 [추정] 전체 현지화 비용의 10~20% 수준.
- KOTRA 설문에서 수출 중소기업 애로 1위가 '바이어 발굴'(22.5%)이지만, 바이어 발굴 이전에 '현지 선호에 맞는 제품·패키지 구성'이 선행 과제임을 현장 관계자들이 지적한다[^3][추정].
- 한국디자인진흥원에 등록된 산업디자인전문회사는 18,014건(2024)이나, 수출기업이 이 DB를 직접 검색해 적합 디자인사를 찾는 원스톱 서비스는 현재 없다.

**표 2.** 현재 수출 패키지 현지화 프로세스의 문제

| 단계 | 현황 | 문제 |
|:---|:---|:---|
| 타깃국 선호 파악 | 무역관 보고서 수동 검색 | 264개국 분량·비정기 업데이트·비전문가 해석 어려움 |
| 디자인 가이드 작성 | 내부 직원 또는 외주 리서치 | 시간 2~4주, 비용 수백만 원 이상 [추정] |
| 디자인사 발굴 | 개인 인맥, 포털 검색 | 수출 패키지 전문성 검증 불가 |
| 디자인사 선정 | 포트폴리오 수동 비교 | 수출 실적·현지화 역량 정보 없음 |

### 2-2. 활용 적정성 4요소

| 요소 | 내용 |
|:---|:---|
| **활용분야** | 수출 중소기업의 타깃국 패키지 현지화 기획·디자인사 매칭 / 무역 지원 기관(KOTRA 무역관)의 기업 컨설팅 보조 도구 |
| **활용빈도** | 수출 기업별 평균 신규 국가 진출 시 1회 + 연 1회 가이드 갱신 / 개별 사용자 기준 주 1~2회 쿼리 [추정] |
| **활용범위** | 전국 수출 중소기업 95,905개사; 특히 소비재(식품·뷰티·생활용품) 수출기업 집중 타깃 / KOTRA 무역관 직원 300명+ 보조 도구로 확장 가능 |
| **중요성** | 수출 생존율 제고(1년 내 중단 50%→ 감소) + 중소기업 현지화 비용 절감 + 산업디자인 전문회사 수주 기회 확대 → 산업부 수출 촉진·산업 생태계 양측에 효과 |

### 2-3. 경영혁신·창업학적 프레임워크

**JTBD(Jobs To Be Done) 관점**: 수출 중소기업이 '패키지 디자인사에 의뢰하는' 행위의 진짜 목적(Job)은 "현지 소비자에게 낯설지 않고 신뢰감을 주는 포장으로 판매를 성사시키는 것"이다. 현재 그 Job은 비용(수백만 원)·시간(2~4주)·정보 비대칭(어느 디자인사가 수출 현지화를 잘하는지 모름) 때문에 미충족 상태다. 글로벌 패키지는 정보 비대칭과 탐색 비용을 공공데이터 기반으로 제거한다.

**Christensen 파괴적 혁신 관점**: 패키지 현지화 컨설팅은 대기업·수출 전문 대행사만 감당할 수 있는 고비용 서비스였다. 공공 API + LLM 조합으로 '소비재 중소기업'이라는 비소비(non-consumption) 시장을 개척한다. KOTRA 데이터는 이미 공개돼 있으나 이를 즉시 활용 가능한 서비스로 전환한 레이어가 없었다.

**블루오션(Kim·Mauborgne)**: 기존 서비스는 무역 정보(트라이빅·buyKOREA)와 디자인 중개(단순 포털)로 분리돼 있다. 글로벌 패키지는 **무역 데이터 × 디자인 DB × LLM 가이드 생성**을 하나의 플로로 통합해 경쟁자가 없는 교차점을 점한다.

### ④ 창의성·독창성

**핵심 창의성: 무역 데이터 × 디자인 산업 DB의 교차 결합**

13회 수상작인 '나만의 통관·수출 도우미'는 식품 통관 절차 안내에 집중했고, 'shannon'은 데이터 자연어 질의에 집중했다. 글로벌 패키지는 **패키지 디자인**이라는 미개척 현지화 레이어를 타깃으로 한다.

독창성 근거:
1. **공공 무역 분석 → 디자인 명세 변환**: KOTRA 시장 분석 문서에서 색채·레이아웃·문구·인증마크 등 디자인 속성을 구조화 추출하는 LLM 파이프라인은 현존하는 공공서비스에 없다.
2. **공공 디자인사 DB 활성화**: 18,014건의 산업디자인전문회사 DB는 현재 단순 목록으로만 제공된다. 이를 수출품목·타깃국·역량 기반 추천 엔진으로 전환한다.
3. **수출 현지화의 '첫 번째 마일'**: 바이어 발굴 전에 현지 소비자 눈높이에 맞는 패키지를 만드는 것이 선행 과제임에도 어떤 공공서비스도 이를 다루지 않는다.

| 속성 추출 | GPT-4o / Claude 3.5 Sonnet (RAG 컨텍스트 주입) | 패키지 디자인 속성 구조화 JSON 추출 | 패키지 도메인 온톨로지 + 룰 기반 후처리 |
| 디자인사 스코어링 | 필터링 + BM25 + LLM 재순위화 (reranker) | 품목·역량 적합도 계산 | 매칭 피드백 데이터 누적 |
| 가이드 생성 | 템플릿 기반 + LLM 자연어 보완 | 가이드 카드 자동 작성 | 도메인 템플릿 레이어 |

**API 래퍼가 아닌 근거 (§2.1 AI 해자 논증)**
- 단순히 "KOTRA 문서를 LLM에 넣고 요약"하는 것이 아니라, **패키지 디자인 도메인 온톨로지**(색채·문구 금기·인증마크·소재 규제의 구조화 스키마)를 자체 구축하고 LLM 출력을 룰로 교차 검증한다.
- KIDP 디자인사 DB를 수출품목·국가에 맞게 매칭하는 **3-way 매칭 알고리즘**은 단순 API 호출이 아닌 자체 스코어링 로직이다.
- 사용자 피드백(매칭 성사율, 가이드 활용 만족도)이 재학습 데이터로 누적되어 **데이터 네트워크 효과**가 형성된다. 기반 LLM이 교체되더라도 벡터DB, 온톨로지, 매칭 알고리즘, 피드백 데이터는 그대로 유지된다.

**서비스 방법**
- 웹 서비스(SaaS)로 제공. 기업 담당자가 품목명·HS코드·타깃국을 입력하면 3분 내에 패키지 가이드 카드와 디자인사 추천 리스트를 출력한다.
- 가이드 카드는 PDF 다운로드 지원. 디자인사 추천 리스트에는 전문분야·주소·연락처·(추후) 수출 포트폴리오 링크 포함.
- KOTRA 해외시장뉴스 API를 주 1회 이상 갱신하여 최신 트렌드 반영.

### 4-5. 차별화 기술의 구매동인 논증

**① 구매동인 가설**

핵심 구매동인: "현지 소비자에게 어색하지 않은 패키지를 빠르게 만들고 싶다 (단, 비용이 감당 가능해야 한다)."

이 Job은 **must-have**에 가깝다. 패키지 디자인이 현지 금기를 위반하면 제품이 진열조차 안 되거나 소비자 반응이 냉담해져 수출 실패로 이어진다[추정]. 반면 현재 대안(외주 리서치 2~4주·수백만 원, 무역관 문의 비정기)은 중소기업이 빠른 의사결정을 내리기 어렵게 한다.

**② 가치 정량화**

| 항목 | 기존 방식 | 글로벌 패키지 | 절감 |
|:---|:---|:---|:---|
| 타깃국 디자인 선호 조사 시간 | 2~4주 [추정] | 3분 이내 | ~99% 단축 |
| 디자인 현지화 리서치 외주 비용 | 100만~500만 원/국가 [추정] | 월 3만~10만 원(구독) | 90%+ 절감 [추정] |
| 디자인사 탐색·선정 시간 | 1~2주 [추정] | 즉시 추천 리스트 | ~95% 단축 |

> 위 수치는 [추정]이며, 파일럿 기업 인터뷰로 검증 예정.

**③ 외부 근거**
- 수출 중소기업 1년 생존율 49.2%: 현지화 실패가 주요 원인 중 하나로 언급[^1].
- KOTRA 수출 중소기업 애로 조사 '바이어 발굴' 22.5%, '해외마케팅' 18.1%[^3].
- 한국디자인진흥원 산업디자인전문회사 18,014개사 DB(2024): 공공 개방 중이나 수출 매칭 서비스에 연결된 사례 없음[^6].

**④ 반증·대안 위협**
- **위협 1 — 기존 관성**: 수출 기업 담당자가 이미 알고 있는 디자인사에 계속 의뢰할 수 있다. → 대응: 3건 무료 체험으로 탐색 비용 제거.
- **위협 2 — LLM 환각**: AI 가이드가 틀린 정보를 줄 수 있다. → 대응: KOTRA 출처 명시 + 온톨로지 교차 검증 + 전문가 검토 경고 표시.
- **위협 3 — KOTRA 직접 서비스 가능성**: KOTRA가 직접 유사 기능을 만들 수 있다. → 대응: KIDP 매칭 레이어는 KOTRA 단독으로는 구현 동기가 약하며, 본 서비스를 KOTRA 채널에 연계하는 협력이 오히려 현실적.

1. **수출 중소기업 역량 평등화**: 대기업만 누리던 현지화 컨설팅을 중소기업도 저비용으로 이용 가능.
2. **산업디자인 산업 활성화**: 18,014개 디자인전문회사의 수출 기업 수주 기회를 공공 DB를 통해 확장.
3. **KOTRA 공공데이터 가치 제고**: 수집·개방되었으나 원시 형태로 낮은 활용도에 머물던 국가정보·해외시장뉴스 데이터를 실제 기업 의사결정 도구로 전환.
4. **수출 생태계 선순환**: 현지화 성공 → 수출 지속 → 고용 유지 → 지역 제조업 활력.
