## A. 활용 산업통상자원부 공공데이터 (탈락요건 충족 필수)

**표 A-1.** 핵심 활용 산업부 공공데이터셋 — **data.go.kr 실재 여부 검증 완료**

| # | 데이터셋명 | 제공 기관 | 데이터셋 ID | data.go.kr URL | 형식 | 갱신 주기 | 활용 내용 | 검증 |
|:---:|:---|:---|:---:|:---|:---:|:---:|:---|:---:|
| 1 | 발전원별 발전량 현황 | 전력거래소(KPX) | 15142651 | https://www.data.go.kr/data/15142651/openapi.do | Open API (JSON) | 실시간 | 발전믹스(원자력·석탄·가스·신재생 등 발전량·비율) 실시간 시각화 및 퀴즈 컨텍스트 생성. 원자력은 발전원의 한 항목(국가 단위 발전량, 호기별 아님)으로 다룸 | 실재 확인 |
| 2 | 현재전력수급현황 | 전력거래소(KPX) | 15056640 | https://www.data.go.kr/data/15056640/openapi.do | Open API (JSON) | 5분 단위 | 공급능력·수요·예비율을 실시간 대시보드·퀴즈 상황변수로 활용 | 실재 확인 |
| 3 (선택 보강) | 계통한계가격(SMP) | 전력거래소(KPX) | 15076302 | https://www.data.go.kr/data/15076302/openapi.do | Open API (JSON) | 실시간/일별 | 발전믹스 변화와 전력 가격의 관계를 설명하는 심화 퀴즈·시나리오에 활용 | 실재 확인 |

> 주: 위 데이터셋은 모두 산업통상자원부 산하 기관(전력거래소) 소관이며, **data.go.kr 실재 여부를 확인**한 Open API다. 핵심 데이터(15142651·15056640) 2종만으로 발전믹스·전력수급 실시간 서비스가 완결되며, SMP(15076302)는 심화 콘텐츠용 선택 보강이다. 본 데이터 구성으로 탈락요건을 충족한다.
>
> **정직성 메모**: 이전 초안에 포함됐던 "한국수력원자력 원자력발전소 실시간 발전현황(15001078)"은 data.go.kr에서 존재가 확인되지 않아(404) 전면 제거했다. 개별 원전 호기 단위 실시간 데이터는 사용하지 않으며, 원자력은 발전원별 발전량(15142651)의 한 항목(국가 단위)으로만 다룬다.
