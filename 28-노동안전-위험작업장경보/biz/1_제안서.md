# 세이프워크(SafeWork) — 감전·가스사고 데이터 기반 위험작업장 예방경보

> 아이디어 간략 개요 (3줄 이내)

감전 및 가스 사고는 특정 작업 유형·장소에서 반복 발생하지만, 현장 근로자와 사업주는 그 위험 패턴을 사전에 파악하기 어렵다. 세이프워크는 한국전기안전공사(KESCO)·한국가스안전공사(KGS)의 사고 이력 공공데이터를 AI로 분석하여, 작업장 단위로 위험도를 예측·경보하고 맞춤형 예방 체크리스트를 제공하는 서비스다. 사고가 일어난 뒤 조사하는 사후 체계에서 벗어나, 데이터가 축적될수록 예측 정확도가 높아지는 선제 예방 경보 체계로 전환한다.

**핵심 기술·서비스·정보 명칭**
- 위험작업장 예측 스코어링 엔진 (AI 패턴 분석 기반 — XGBoost·K-Means 군집화)
- 작업 유형별 예방 체크리스트 자동 생성 (RAG + 도메인 특화 규칙 베이스)
- 사고 이력 히트맵 및 위험도 경보 대시보드 (웹·모바일 반응형)

---

## 1. 아이디어 기획 핵심내용 (구체성, 우수성)

### 1.1 무엇을 만드는가

세이프워크는 **산업 현장의 감전·가스 사고 예방을 위한 위험도 예측 경보 플랫폼**이다. 세 가지 핵심 기능으로 구성된다.

**① 위험작업장 스코어링 대시보드**

장소별 감전사고 이력 데이터(KESCO, 데이터셋 15119870)와 가스사고 현황 데이터(KGS, 15067796), 국내 가스시설 현황 데이터(KGS, 15067840)를 결합하여, 지역·업종·작업 유형 단위로 위험 점수를 산출하고 지도·목록 형태로 시각화한다. 사업주·안전관리자·현장 근로자가 자신의 작업장 유형을 조회하면 "이 유형의 작업장에서 최근 N년간 감전 사고가 M건 발생했으며, 원인 상위 3위는 △△△"라는 형태의 정보를 즉시 확인할 수 있다. 다중이용시설 전기안전점검 결과(KESCO, 15159822)·안전등급 이력정보(KESCO, 15145904)를 보조 변수로 결합하여 점검 부적합 이력이 있는 시설의 위험도를 보정한다.

**② AI 기반 위험도 예측 및 경보**

사고 원인(작업 종류·설비 상태·시간대·계절)과 발생 장소 유형을 학습한 XGBoost/Random Forest 모델이 "현재 유사한 조건을 가진 작업장의 예상 위험 등급(저/중/고/위험 4단계)"을 산출한다. 특정 계절·작업 시기가 도래하거나, 신규 가스시설 가동 현황 데이터가 갱신될 때 자동으로 경보를 발송한다. 경보 채널은 웹 알림·이메일(Phase 1), SMS·카카오 알림톡 연동(Phase 2)으로 단계적으로 확장한다[^추정1]. SHAP(SHapley Additive exPlanations) 기반 설명 가능 AI(XAI)를 적용하여 "이 점수가 높은 이유: 밀폐 공간 + 여름 + LPG 시설"을 인간이 읽을 수 있는 언어로 제시한다.

**③ 맞춤형 예방 체크리스트 자동 생성**

작업장 유형(건설·제조·서비스업 등)과 주요 설비(LPG 시설 보유 여부, 고압 전기설비 여부 등)를 입력하면, 해당 작업 유형의 사고 원인 상위 항목과 연계된 점검 항목이 RAG 파이프라인(벡터 DB + LLM 보조)으로 자동 생성된다. 생성된 체크리스트는 PDF 출력·모바일 확인이 가능하며, 산업안전보건법 법정 점검 조항이 각 항목에 자동 연결된다. 체크리스트 완료 여부를 저장하여 정기 이력 관리가 가능하다.

### 1.3 왜 지금인가 (구체성·우수성)

기존 예방 체계는 **사후 조사 및 정기 방문점검** 중심이다. 전기안전공사는 연 1~2회 방문점검을 수행하고, 가스안전공사도 정기 검사 위주로 운영된다. 사고 이력 데이터가 공공데이터포털에 이미 개방되어 있음에도, 이를 현장 수준의 예방 경보로 전환하는 서비스는 존재하지 않는다.

세이프워크의 우수성은 "분산된 사고 이력을 패턴화하여 선제 경보로 전환한다"는 데 있다. 사고는 무작위가 아니라 **특정 작업 유형·장소 조건에서 반복**된다. 중대재해처벌법(2022년 시행, 2024년 5인 이상 사업장 확대)은 법적 의무화로 소규모 사업장의 안전 솔루션 수요를 직접 창출하고 있어, 서비스 도입 시기로 최적이다.[^5]

## 2. 아이디어 구상 및 제안배경 (활용적정성)

### 2.1 해소하는 사회문제 — 반복되는 감전·가스 산업재해 사망

**산업 현장 감전·가스 사고는 매년 반복적으로 사망자를 낳는다.**

고용노동부 산업재해 통계에 따르면, 2023년 전체 산업재해 사망자 수(사고사망 기준)는 598명이다.[^1] 이 중 감전 사고 사망자는 국내에서 매년 20~30명 수준으로 집계되며[^6], 전기 관련 재해는 전체 산재의 1~2%를 차지하지만 사망률(치명도)이 평균보다 높다.[^7] 가스 사고의 경우, 한국가스안전공사 공공데이터(15067796) 기준으로 연간 수십 건 이상 발생하며 인명피해(사상자)가 동반된다.[^2]

**사고의 집중성과 반복성이 핵심이다.**

KESCO 장소별 감전사고 현황(15119870)을 분석하면 공장·건설현장·상업시설 등 특정 장소 유형에서 사고가 집중되는 패턴이 뚜렷하다. 동일한 작업 유형(전기 배선·수리, LPG 용기 교체, 배관 용접, 밀폐 공간 용접 등)에서 반복적으로 발생한다. KGS 가스사고 현황(15067796)에서는 겨울철(난방 수요 증가)과 여름철(고온 환경·냉방기기 가스 누출) 사고 집중 패턴이 확인된다. 이는 시기와 작업 유형이 일정하면 **예방이 가능한 사고 유형**임을 뜻한다.

**현 예방 체계의 구조적 공백:**

| 주체 | 현황 | 한계 |
|:---|:---|:---|
| 한국전기안전공사(KESCO) | 정기 방문점검, 전기안전여기로(민원 포털) | 사후 점검·민원 중심; 사고 패턴의 작업장 단위 경보 없음 |
| 한국가스안전공사(KGS) | 정기 가스시설 검사, 사고 원인 조사 | 검사 중심; 유사 조건 사업장에 선제 경보 없음 |
| 고용노동부 | 산업재해 조사 보고 | 사고 이후 조사; 예방 경보 미제공 |
| 현장 사업주·근로자 | 법정 안전교육 이수 | 자신의 작업 유형 관련 사고 통계를 실시간 조회 불가 |

특히 소규모 사업장(50인 미만)은 전문 안전관리자를 두기 어렵고, 안전 정보 접근성이 낮다. 사고 이력 공공데이터는 이미 개방되어 있지만, 이를 "우리 작업장이 위험한가"라는 질문에 답해주는 형태로 가공하는 서비스가 없다.

### 2.2 활용분야·활용빈도·활용범위·중요성

**활용분야:** 산업안전 관리, 중소 사업장 자율 안전점검, 전기·가스 관련 공공기관 예방 업무, 산업재해 보험 위험도 평가

**활용빈도:** 상시 조회(대시보드), 작업 전 체크리스트 확인(일 단위), 계절별 위험 경보 자동 발송(주/월 단위), 연간 안전 리포트 생성(연 단위)

**활용범위:** 전국 모든 전기·가스 설비 보유 사업장. 한국가스안전공사 국내 가스시설 현황(15067840) 기준 전국 LPG·도시가스 시설 보유 사업장 수백만 개소, KESCO 다중이용시설 전기안전점검 결과(15159822)상의 점검 대상 시설 전체가 잠재적 서비스 대상이다.[^3]

**중요성:** 산업재해 사망은 개인과 가족의 비극이자 사회적 비용이다. 안전보건공단(KOSHA) 추정에 따르면 산재 사망 1인당 경제적 비용(의료비·법적 배상·생산성 손실 포함)은 수억 원에 달한다.[^4][추정] 감전·가스 사고는 반복 패턴이 뚜렷한 예방 가능 유형이며, 선제 정보 제공만으로도 상당한 감소 효과를 기대할 수 있다. 중대재해처벌법 시행 이후 5인 이상 사업장에 법적 안전관리 의무가 부과되어, 저비용 자율 안전 솔루션에 대한 구조적 수요가 발생하고 있다.[^5]

- **15067796 (가스사고 현황):** 월별·원인별(시설 결함·취급 부주의·공급자 과실 등) 사고 건수 및 인명피해. 계절별(겨울·여름 집중 패턴)·원인별 패턴 분석에 활용하여 시기별 경보 트리거를 설계한다.
- **15067840 (국내 가스시설 현황):** 시도별 LPG/도시가스 시설 수. 시설 밀도 대비 사고 발생률 계산(노출량 보정)에 활용하여 시설이 많은 지역의 단위 위험도를 실질적으로 산출한다.
- **15159822 (다중이용시설 전기안전점검 결과):** API 형식으로 제공되는 점검 부적합 이력. 부적합 판정 시설을 위험도 스코어 보정 변수로 사용하여 점검 불합격 이력이 있는 사업장의 위험 등급을 상향 조정한다.
- **15145904 (안전등급 이력정보):** 자가용 전기설비 안전등급 이력. 등급 하락 이력이 있는 설비의 위험도를 가중 반영한다.

- **① 구매동인 가설:** 본 서비스의 핵심 차별점(사고 이력 기반 작업장 단위 경보·체크리스트)이 건드리는 고객 의사결정 요인은 "중대재해처벌법 위반 시 형사처벌 회피"와 "사망 사고 발생 시 손해배상 리스크 최소화"다. 이는 **must-have** 영역이다 — 법적 의무화로 비구매(사용 안 함) 자체가 법적 위험을 의미하기 때문이다. [추정: 법적 의무가 직접 구매동인임은 검증, 이 서비스가 의무 이행 수단으로 충분한지는 파일럿에서 검증 예정]

- **② 크기 정량화:** 중대재해 1건 발생 시 사업주 형사처벌(1년 이상 징역)·손해배상 수억 원[^4][추정]의 리스크와 비교하면, 월 9,900~29,900원의 구독료는 전환비용 대비 압도적으로 낮다. 10배 규칙(가치/비용 ≥ 10배)을 충족한다.

- **③ 외부 근거:** 중대재해처벌법 5인 이상 사업장 적용(2024)[^5], KESCO·KGS 공공데이터 감전·가스 사고 집중 패턴(데이터셋 15119870·15067796), 고용노동부 사고사망 598명(2023)[^1]. 파일럿 후 구체 수치로 교체 예정.

- **④ 반증·대안 위협:** 고객이 "KESCO/KGS 정기점검만으로 충분하다"고 판단하면 이탈한다. 대응: 정기점검(연 1~2회)이 커버하지 못하는 일상 작업 전 체크리스트·조건 기반 경보를 무료 기능으로 제공하여 마찰 없이 가입 → 이후 이력 관리·리포트 유료화로 전환.

### 3-④ 창의성·독창성

세이프워크의 창의성은 **"흩어진 사고 이력을 현장 수준의 선제 경보로 연결하는 정보 중개"** 에 있다. 한국전기안전공사와 한국가스안전공사는 수십 년간 사고 데이터를 축적해왔으며 이를 공개했지만, "지금 내가 하려는 작업이 위험한가"라는 실시간 질문에 답해주는 서비스는 존재하지 않는다.

독창성의 두 번째 축은 **두 기관 데이터의 교차 결합**이다. 감전 사고와 가스 사고는 상호 독립적이지 않다 — 용접 작업은 감전(아크 전류)과 가스(아세틸렌·LPG) 위험을 동시에 내포한다. 이를 통합적으로 분석하는 서비스는 현재 없다. 세 번째 축은 **SHAP 기반 설명 가능 AI** 적용으로 "스코어가 높은 이유"를 현장 언어로 번역하여, 비전문가 사업주도 경보의 근거를 이해하고 즉각 행동할 수 있게 한다는 점이다.

| AI 모델 — 체크리스트 생성 | RAG 파이프라인 (벡터 DB + LLM 보조) | 사고 원인 설명 텍스트 → 예방 조치 체크리스트 변환 |
| XAI | SHAP (SHapley Additive exPlanations) | 위험 스코어 근거 설명 ("밀폐 공간 + 여름 + LPG → 위험 高") |
| 백엔드 | FastAPI (Python) | REST API 서버, 경보 스케줄러 |
| 프론트엔드 | React + Leaflet.js (지도) | 대시보드, 히트맵 시각화, 반응형 UI |
| 알림 | 이메일(SMTP) / 웹 푸시 (Phase 1); SMS·카카오 알림톡 (Phase 2) | 위험 경보 발송 |

**AI 해자 논증 (단순 API 래퍼가 아닌 이유):**

세이프워크의 AI는 공공 LLM API를 단순 호출하는 래퍼가 아니다. 독자적 가치는 세 층위에 있다:

1. **독자 도메인 데이터:** KESCO·KGS 사고 이력 5종 데이터셋 자체가 독점적 학습 입력이다. 이 데이터로 학습된 XGBoost 위험도 예측 모델은 범용 LLM이 대체할 수 없다. 사고 원인 변수(장소 유형·작업 종류·계절·시설 상태)와 인명피해 간의 통계적 관계를 도메인 데이터로 학습한 것이 핵심 자산이다. 사용자가 체크리스트를 완료할 때마다 쌓이는 점검 이력은 데이터 네트워크 효과를 만든다.

2. **버티컬 워크플로 통합:** 데이터 수집 → 전처리 → 스코어 산출 → 경보 발송 → 체크리스트 생성 → 이력 관리까지 전 과정이 단일 파이프라인으로 연결된다. 단발 생성이 아닌 업무 전 과정(작업 전 → 점검 중 → 이력 보관)에 끼어드는 워크플로 통합이다.

3. **모델 교체가능성 전제:** LLM(체크리스트 생성 보조)이 더 좋은 모델로 교체되더라도, 세이프워크의 핵심 자산인 사고 이력 데이터 파이프라인·XGBoost 위험도 스코어 모델·도메인 특화 예방 규칙 베이스는 그대로 유지된다. 기반 LLM 상품화에 의존하지 않는 구조다.

### 4.3 경영혁신·창업학적 프레임워크

**Christensen 파괴적 혁신 (Disruptive Innovation):**

현재 산업안전 서비스는 대기업 대상 전문 안전 컨설팅(건당 수백~수천만 원, 전문인력 필수)이 주류다. 세이프워크는 공공데이터와 AI를 활용하여 소규모 사업장이 낮은 비용(월 9,900~29,900원)으로 접근할 수 있는 "로엔드 혁신" 경로를 연다. 기존 컨설팅의 비소비자(Non-consumer)였던 50인 미만 사업장을 새로운 고객으로 포섭한다는 점에서 파괴적 혁신의 전형적 패턴에 해당한다.

**JTBD (Jobs To Be Done):**

고객의 실제 Job: "우리 작업장에서 사고가 날까봐 걱정되는데, 어디서부터 점검해야 할지 모르겠다. 중대재해처벌법 위반으로 처벌받고 싶지 않다." → 세이프워크는 이 Job을 "지금 당장 내 작업 유형의 위험 스코어를 보고, 오늘 점검할 항목을 체크리스트로 받아 저장한다"로 해결한다.

**Kim·Mauborgne 블루오션 (Blue Ocean Strategy):**

기존 경쟁 구도(KESCO 방문점검 ↔ KGS 정기검사)는 사후·공급자 중심으로 경쟁이 고착되어 있다. 세이프워크는 경쟁이 없는 영역, 즉 "사전·수요자(사업주) 중심 자율 예방 경보"라는 블루오션을 개척한다. 가치 곡선의 비교 축을 전문인력 필요도(제거)·방문 대기시간(제거) → 즉시 조회·체크리스트 자동화·법령 연결(증가)로 재설계한다.
