## 3. 추가 조사 예정 항목 (파일럿 후 확충)

- 감전·가스 사고별 작업 유형 세분화 통계 (KESCO·KGS 내부 보고서)
- 국내외 안전 경보 시스템 도입 효과 연구 (KCI·RISS 논문)
- 중대재해처벌법 시행 이후 소규모 사업장 안전 투자 변화 통계 (고용노동부 보도자료)
- 산업안전 SaaS 시장 규모 자료 (IBK경제연구소·중소벤처기업부 보고서)
- 해외 유사 서비스(미국 OSHA 사고 데이터 활용 사례, EU ESENER 조사) 벤치마크
- XGBoost/RandomForest 위험도 예측 모델 정밀도·재현율 파일럿 측정값
- 사용자 파일럿 50개 사업장 피드백 데이터 (Phase 1 완료 후)
