## D. 데이터 정직성 메모

- **날조 없음**: 본 연구 파일에 기재된 모든 URL은 data.go.kr 또는 해당 기관 공식 포털의 실재 주소이다.
- **[추정] 항목**: 지역 청년 수도권 취업 의향 62%, 취업포털 시장 8,000억, 채용 비용 300~500만 원, 청년 탐색 기간 3~6주 수치는 직접 출처 재확인 전까지 추정으로 표기.
- **[확인필요]**: KICOX 고용동향 공공데이터 포털 등록 번호·API 엔드포인트 주소는 공식 포털(https://www.kicox.or.kr/publicData/publicDataList)에서 직접 확인 후 업데이트 필요.
- 출처 수량 현황: **9개 / 목표 1,000개** — 초안 단계이므로 파일럿 운영 및 추가 조사를 통해 확대 예정.
