# 솔라체크 — 우리집·우리건물 지붕 태양광 발전 잠재량·수익성 자가진단

주소(또는 건물 위치)를 입력하면, 산업통상자원부 산하 전력거래소(KPX)의 지역별 시간별 태양광 발전량 실측 공공데이터(15103243)와 한국에너지공단의 기초지자체별 신재생 보급현황(15086292)을 결합하여, 해당 건물 지붕에 설치할 경우 예상 발전량·투자회수기간·연간 절감액·탄소 감축량을 즉시 산출하는 자가진단 서비스다. 설치업체 영업 견적이 아닌 **공공 실측 데이터 기반 중립 분석**이므로, 시민·건물주가 업체 방문 전에 객관적 타당성을 먼저 파악할 수 있다.

**핵심 기술·서비스·정보 명칭**
- 지역 실측 기반 지붕 발전량 예측 엔진 (KPX 시간별 태양광 발전량 + 지역별 보급현황 결합)
- 투자수익성 자가진단 모듈 (설치비용 범위 입력 → IRR·회수기간·절감액 자동 산출)
- 탄소 감축 기여 계산기 (지역 발전 믹스 기반 CO₂ 대체량 환산)
- 지자체·아파트 단지 단위 집단 진단 대시보드 (지역사회 참여 채널)
- 설치 적합성 지수 (경사·면적·그늘 자기신고 → 보정 계수 적용)

## 1. 아이디어 기획 핵심내용 (구체성, 우수성)

### 1.1 무엇을 만드는가

"솔라체크"는 세 가지 핵심 기능을 하나의 웹·앱 서비스로 제공한다.

**① 지붕 발전량 예측 — 공공 실측 데이터 기반**

사용자가 주소와 지붕 면적(㎡), 경사 방향(남·동·서), 주요 그늘 유무를 입력하면, KPX 지역별 시간별 태양광 발전량 공공데이터(15103243)에서 해당 시·군·구 최근 3년 시간별 실측치를 가져온다. 기상청 일사량 보정 계수[^1]와 한국에너지공단 기초지자체별 신재생 보급현황(15086292)의 지역별 평균 설비이용률을 결합하여, 1kW 설치 시 연간 발전량(kWh/kWp)을 산출한다. 전국 평균은 약 1,100~1,300kWh/kWp 수준이나, 지역·경사별 편차가 최대 20~30% 이상 발생하므로[^2] 지역 실측치 적용이 핵심이다.

**② 투자수익성 자가진단**

설치 용량(kW)을 선택하고, 현재 사용 중인 전기요금제(가정용/일반용/산업용)를 입력하면 아래 항목을 자동 산출한다.

| 출력 항목 | 산출 방법 |
|:---|:---|
| 연간 발전 예측량(kWh) | 지역 실측 시간별 발전량 × 설비이용률 보정 |
| 연간 절감 전기요금(만원) | 발전량 × 한전 계약종별 전력단가(KEPCO 공시 기준) |
| 잉여전력 판매 수입(만원) | 발전량 중 자가소비 초과분 × 소규모 태양광 SMP+REC 단가[^3] |
| 초기 투자비 범위(만원) | 용량별 국내 설치비 통계(에너지공단 신재생 보급현황 단가 참조)[^4] |
| 단순 투자회수 기간(년) | 총비용 ÷ (절감+판매 수입) |
| IRR(%) | 20년 현금흐름 할인 기준 내부수익률 |
| CO₂ 감축량(tCO₂/년) | 발전량 × 한국 전력 배출계수 0.4594 kgCO₂/kWh[^5] |

**③ 지역·단지 집단 진단 대시보드**

개인 진단 결과를 익명 집계하면 지자체별·아파트 단지별 "지붕 태양광 잠재 지도"가 형성된다. 지자체 담당자나 아파트 관리사무소는 집단 진단 결과를 바탕으로 집합 입찰을 추진해 설치단가를 인하할 수 있다. 에너지공단의 기초지자체별 신재생 보급현황 데이터와 대조하면, 보급 목표 대비 실제 달성률·잠재량 갭을 시각화할 수 있다.

### 1.3 AI 해자 논증 — API 래퍼가 아닌 이유

본 서비스의 AI/예측 레이어는 단순 LLM 호출이 아니라 아래 독자 자산 위에 구축된다.

- **도메인 실측 데이터 자산**: KPX의 수년치 지역별 시간별 태양광 실측값은 수십만 개 시계열 레코드다. 이를 지역·계절·기상 조건별로 클러스터링하면 설치 전 건물의 기대 발전량을 사후 실적과 대조·보정할 수 있다. 이 데이터 파이프라인은 타 서비스가 쉽게 복제하지 못하는 독자 가치다.
- **피드백 루프 데이터 네트워크 효과**: 사용자가 설치 후 실제 발전량을 앱에 입력하면, 그 데이터가 같은 지역·건물 유형의 다음 예측 정확도를 높인다. 사용자가 늘수록 예측 모델이 정교해지는 **데이터 네트워크 효과**가 발생한다.
- **모델 교체가능성 전제**: 기반 LLM(추천 문구 생성 등 일부)은 교체돼도 핵심 가치(지역 실측 기반 발전량·수익성 예측)는 독립적으로 작동한다. 예측 엔진과 DCF 계산 모듈은 LLM 없이도 완전히 동작한다.
- **버티컬 워크플로 통합**: 진단 → 집합 입찰 → 보조금 신청 가이드 → 설치 후 발전량 모니터링의 전 과정을 커버하는 워크플로가 서비스의 해자다.

## 2. 아이디어 구상 및 제안배경 (활용적정성)

### 2.1 사회 문제 현황 — 통계 기반

한국의 재생에너지 보급은 선진국 대비 현저히 낮은 수준에 머물러 있다.

- **신재생 발전 비중**: 2023년 국내 전체 발전량 중 신재생에너지 비중은 약 9.9%[^6](한국에너지공단 신재생에너지 보급통계). 2030년 정부 목표인 30.2%[^7](제10차 전력수급기본계획)에 도달하려면 10년 내 발전 비중을 3배 이상 늘려야 한다.
- **태양광 보급 속도 정체**: 국내 태양광 누적 설치용량은 2023년 말 기준 약 26.3GW[^8](에너지공단 신재생에너지 보급통계)이나, 2023년 신규 설치량은 전년 대비 감소세로 전환되었다. 보급 확대의 핵심 걸림돌로 "설치 전 수익성 정보 접근 어려움"이 지목된다[^9][추정: 에너지공단 주민수용성 연구 인용 필요].
- **주택·건물 지붕 미활용 잠재량**: 국내 주거·상업 건물 지붕의 태양광 설치 가능 잠재량은 약 42~96GW[^10][추정: 에너지경제연구원 보고서 추정치 범위]로 추산되며, 이 중 실제 설치된 비율은 매우 낮다.
- **정보 비대칭 문제**: 태양광 설치를 고려하는 일반 건물주가 수익성을 미리 파악할 수 있는 중립적 공공 서비스가 부재하다. 현재는 설치업체에 직접 연락해 영업 견적을 받는 것이 유일한 방법이며, 이 과정에서 과다 견적·부실 시공 위험이 발생한다[^11].
- **지역별 격차**: 에너지공단 기초지자체별 신재생 보급현황(15086292)에 따르면 지자체 간 1인당 태양광 보급 격차가 10배 이상으로, 정보 접근성의 지역 불균형이 보급 격차를 심화시키고 있다[추정].

**활용분야·빈도·범위·중요성**

| 항목 | 내용 |
|:---|:---|
| **활용 분야** | 주택 소유자·건물주의 태양광 설치 의사결정 지원, 지자체 재생에너지 보급 계획 수립, 에너지 복지(가계 전기요금 절감), 탄소중립 이행 기여 |
| **활용 빈도** | 일반 사용자: 설치 의사결정 시 1회 이상(연간 수십만 건 신규 진단 예상). 지자체 담당자: 월 1회 보급 현황 대시보드 조회. 설치 후 사용자: 월 1회 발전 실적 모니터링 |
| **활용 범위** | 전국 기초지자체(228개)별 태양광 발전 실측 데이터 커버. 가정용·상업용·산업용 전 건물 유형 |
| **중요성** | 2030 재생에너지 목표 달성의 핵심 과제인 "분산형 소규모 태양광 확대"는 시민·건물주의 자발적 설치 참여가 전제 조건이며, 참여의 첫 단계는 정보 접근성 해소다. 공공데이터를 활용한 중립 자가진단 서비스는 정보 비대칭을 해소하고 보급 속도를 구조적으로 높이는 공공 인프라 역할을 한다 |

|:---|:---:|:---:|:---:|:---:|

**차별점 종합**: 기존 수단이 "업체 방문 후 영업 정보" 또는 "정부 안내 페이지"로 양분되어 있는 반면, 솔라체크는 공공 실측 데이터를 자동 분석하여 **중립적이고 즉각적인 수익성 수치**를 제공하는 최초의 공공데이터 기반 자가진단 서비스다.

### 3.4 창의성·독창성

- **공공 실측 발전량 + 수익성 자동 연산**: 태양광 자가진단 서비스는 국내에서 선례가 없다. Google Project Sunroof와 유사하나, 위성 일사량 추정 대신 KPX의 **실제 전력계통 발전 실측치**를 쓴다는 점이 근본적 차이다. 실측 기반이므로 계절·날씨 변동성이 이미 반영된 값이다.
- **집단 진단 → 집합 입찰 경로**: 개인 진단 데이터를 익명 집계해 지역 잠재량 지도를 만들고, 집합 입찰을 유도하는 플랫폼 효과는 기존 서비스에서 찾을 수 없는 독창적 설계다. 집합 입찰 시 설치단가를 20~30% 낮출 수 있다는 국내외 사례[^12][추정]가 있어 실질적 비용 절감 효과를 기대할 수 있다.
- **예측-실적 피드백 루프**: 설치 전 예측과 설치 후 실제 발전량의 차이를 누적 학습함으로써, 지역·건물 유형별 예측 정확도가 시간이 지날수록 향상되는 **데이터 네트워크 효과**를 설계에 내재한다.

**그림 3.** 사용자 여정과 서비스 엔진 연결 구조

**서비스 단계별 시나리오 (시민 이용자 기준)**

1. **진입** — 포털 검색 또는 지자체 에너지 담당 부서 링크로 접속. 회원가입 없이 주소 입력만으로 이용 가능(로그인은 실적 모니터링 시에만 필요).
2. **입력** — 주소 자동 완성 → 지붕 면적(㎡, 슬라이더), 주요 방향(남/동남/서남), 그늘 여부(예/아니오·정도 선택) → "진단 시작" 클릭.
3. **결과** — 5~10초 내 리포트 표시: 권장 설치 용량(kW), 연간 예상 발전량, 연간 절감 요금, 단순 회수 기간, IRR, CO₂ 감축량, 현재 지역 보급률·전국 평균 비교.
4. **심화 분석** — 설치 용량 범위를 조정하거나 요금제를 변경하면 실시간으로 수익성 재계산. 3개 시나리오(보수/기본/낙관) 비교 표 자동 생성.
5. **다음 단계 안내** — 지역 내 에너지공단 인증 설치업체 목록 링크, 지자체 보조금 신청 페이지 링크, 동네 집합 진단 참여 옵션.

### 4.3 고객 확보 전략 (GTM)

**ICP(이상적 고객 프로필)**
- **1차 타깃**: 단독·다세대 주택 소유자(30~60대), 소규모 상업건물주 — 전기요금 부담이 크고 설치를 고려하나 정보가 없는 층

- **3차 타깃**: 아파트 관리사무소·입주자대표회의 — 공용전기 절감 방안 모색 주체

| 채널 | 전술 | 예상 CAC |
|:---|:---|:---|
| 지자체 에너지 부서 협력 | 시·군·구 에너지 담당자에게 B2G 제안 → 지자체 홈페이지·SNS에 서비스 링크 게재 | 낮음(채널 비용 거의 없음) |

| 검색광고 (SEO/SEM) | "태양광 설치 비용", "태양광 수익성" 등 고관여 키워드 | 2~5만원/진단[추정] |

**퍼널 및 리텐션**
- 인지 → 주소 입력 → 진단 완료 → 결과 공유(SNS 바이럴 공유 유도 → CAC↓) → 설치 후 등록 → 월 발전 실적 확인(월 재방문)
- 설치 후 사용자 리텐션: 월 발전 실적 vs 예측 비교 알림 → 이탈률↓, 피드백 데이터 축적

### 4.4 경영혁신·창업학적 프레임워크

**① Christensen 파괴적 혁신 (Disruptive Innovation)**

기존 태양광 시장에서 수익성 정보는 "설치업체 방문 견적"이라는 복잡하고 접근 장벽이 높은 방식으로만 제공되어 왔다. 솔라체크는 공공데이터를 기반으로 **비소비층(태양광을 고려는 하지만 정보 접근 장벽으로 포기한 대다수 건물주)을 새 시장으로 편입**하는 파괴적 혁신 구조를 갖는다. Christensen의 이론에서 파괴적 혁신자는 "더 단순하고, 더 저렴하고, 더 접근하기 쉬운" 대안으로 비소비층을 활성화한다 — 솔라체크가 정확히 그 역할을 한다[^14].

**② JTBD (Jobs To Be Done) 프레임워크**

건물주의 JTBD는 "태양광을 설치하고 싶다"가 아니라 "업체에 끌려다니지 않고 **객관적 근거로 내 건물에 태양광이 맞는지 먼저 알고 싶다**"이다. 현재 이 Job은 충족 수단이 없어 진행이 막힌다. 솔라체크는 이 핵심 Job을 3분 내에 충족시킴으로써, 기존 업체 의존형 프로세스를 역전한다.

**③ 블루오션 전략 (Kim·Mauborgne)**

설치업체와 경쟁하지 않는다. 중립적 공공 정보 제공이라는 새 가치곡선을 그려 **비경쟁 공간**에 포지셔닝한다. 설치업체는 오히려 고품질 리드를 확보하는 파트너가 된다 — 경쟁이 없는 동반 성장 구조다.

### 4.5 차별화 기술의 구매동인 논증

**① 구매동인 가설**

본 서비스의 핵심 차별점("공공 실측 데이터 기반 중립 자가진단")이 건드리는 고객 의사결정 요인: **"설치 전에 업체를 믿을 수 없는 상황에서, 신뢰할 수 있는 객관적 수치가 있으면 의사결정을 할 수 있다."**

이 요인은 **must-have에 가까운 준필수 요소**다. 수익성 정보가 없으면 설치 결정 자체를 내리지 못한다(포기 또는 무기한 연기). 반면 중립 정보가 주어지면 의사결정 임계를 낮출 수 있다.

**② 가치 정량화**

- 시간 절약: 업체 방문 견적 프로세스 평균 2~4주(업체 선택 → 현장 방문 일정 조율 → 견적서 수령 → 비교) → 솔라체크 3분으로 단축. **약 2~4주 시간 절감[추정]**.
- 비용 절약: 집합 입찰 진행 시 개별 견적 대비 설치단가 20~30% 인하[^12][추정]. 3kW 설치 기준 약 120~200만원 절약[추정].
- 오류 위험 감소: 과다 견적·부실 시공 피해를 사전에 방지. 연간 소비자 피해 신고 건수[^15][추정: 한국소비자원 태양광 관련 피해 통계 확인 필요].

**③ 외부 근거**

국내 소비자들의 태양광 설치 포기 이유 1위는 "수익성 불확실"[^9][추정: 에너지공단 주민수용성 연구]. 에너지공단 기초지자체별 신재생 보급현황 데이터는 지자체별 보급 격차가 실재함을 보여주며, 이 격차의 원인 중 하나가 정보 접근성 차이임을 선행 연구들이 지목한다[추정].

**④ 반증·대안 위협**

- **업체가 직접 무료 견적 제공**: 설치업체는 이미 무료 현장 견적을 제공하나, "중립성 부재"라는 근본 문제가 해소되지 않는다. 오히려 업체 방문 전 중립 진단이 있어야 견적의 신뢰성을 검증할 수 있다.
- **정부가 직접 만들 수 있음**: 에너지공단이 유사 서비스를 구축할 가능성이 있으나, 공공기관의 개발·운영 속도는 민간에 비해 느리고, 사용자 경험(UX) 최적화가 약하다는 구조적 한계가 있다. 솔라체크는 선점 효과와 데이터 축적으로 방어한다.
- **정보가 있어도 실제 설치로 이어지지 않을 수 있음**: 이 위협은 실재한다. 진단→설치 전환율이 낮을 경우 리드 수익 모델의 전제가 흔들린다. 완화 전략: 집합 입찰 경로로 개인 단위보다 높은 전환율을 유도하고, 지자체 SaaS 구독 모델로 리드 의존도를 낮춘다.

본 제안서의 모든 통계·수치에는 출처 각주를 부기하였다. 검증된 공식 통계 출처가 없는 항목은 **[추정]** 으로 명시하였으며, 추정값과 공식 수치를 혼용하지 않았다. 산업통상자원부 산하 기관(전력거래소·한국에너지공단) 공공데이터셋 ID는 data.go.kr에서 실재 확인된 것만 기재하였고, 새 ID를 생성하거나 지어내지 않았다. 참고문헌 중 [추정] 표기된 항목은 5_research/ 조사 확대를 통해 실제 출처로 대체할 예정이다.
