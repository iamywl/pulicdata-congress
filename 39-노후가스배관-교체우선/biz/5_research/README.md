## 조사 메모

- **탈락요건 충족**: KGS(한국가스안전공사)는 「가스안전관리법」 제28조에 따라 산업통상자원부 장관이 설립 허가한 공공기관. 따라서 KGS 데이터(15067840·15067796)는 "산업통상자원부 공공데이터" 탈락요건을 충족한다.
- **데이터 품질 주의**: 가스시설 현황(15067840) 데이터에 배관 설치연도 컬럼이 포함돼 있는지 실제 다운로드 후 확인 필요. 미포함 시 설치연수 대신 최종 점검일·시설 등록일로 대체 산정하는 방법론 수정 필요.
- **추가 데이터 가능성**: 가스공급사(도시가스 33개사)가 자체 보유한 배관 GIS 데이터는 공개되지 않으나, 협력 MOUs 체결 시 결합 가능. 이 경우 공간 정밀도가 크게 향상됨.
