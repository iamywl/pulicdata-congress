# 빌딩에코 (BuildingEco)
## — 에너지 다소비 건물 효율 등급·개선 컨설팅 플랫폼

> **아이디어 간략 개요 (3줄 이내)**
> 한국전력 계약종별·산업분류별 전력사용량 공공데이터와 에너지공단 100대 에너지통계를 결합해, 건물주가 자신의 건물 에너지 효율이 동종·동급 건물 대비 어느 위치인지를 즉시 파악할 수 있는 벤치마크 대시보드를 제공한다.
> AI 기반 이상치 탐지로 과소비 원인을 진단하고, 단열·설비 개선·에너지 절약 시나리오별 투자 회수기간(PBP)을 자동 산출해 건물주가 우선순위 결정을 내릴 수 있도록 돕는다.
> 장기적으로는 탄소중립 빌딩 인증·그린리모델링 지원금 연계로 수익화하며, 중소 빌딩(연면적 500~3,000 m²) 약 22만 동을 1차 타깃으로 사업화한다[추정].

**핵심 기술·서비스·정보 명칭**
`에너지 효율 벤치마크(Energy Efficiency Benchmarking)` · `건물 에너지성능 이상치 탐지(Anomaly Detection on Building EUI)` · `개선 우선순위 추천(ROI-ranked Retrofit Recommendation)`

## 1. 아이디어 기획 핵심내용 (구체성, 우수성)

### 1-1. 무엇을 만드는가

**빌딩에코**는 건물 에너지 소비 데이터를 공공 API로 자동 수집·정규화하고, 동종 건물군과 비교한 **에너지 사용 강도(EUI, kWh/m²·year) 백분위 등급**을 제공하는 SaaS형 대시보드다.
다음 세 가지 핵심 기능을 갖는다.

| 기능 | 설명 | 핵심 데이터 |
| :--- | :--- | :--- |
| **① 건물 효율 벤치마크** | 계약종·용도·연면적 조건으로 유사 건물군 평균 EUI 대비 내 건물 위치(백분위) 표시 | KEPCO 계약종별 전력사용량 (ID: 15101360) |
| **② 이상치 탐지 & 원인 진단** | AI가 월별 사용량 이상 급등 구간을 탐지, 원인 가설(공조, 조명, 외피 손실) 제시 | KEPCO 산업분류별 전력사용량 (ID: 15101403) |
| **③ 개선 시나리오 PBP 계산** | LED 교체·단열 보강·ESS 도입 등 시나리오별 절감 전력량(kWh)·비용·회수기간 자동 산출 | 에너지공단 100대 에너지통계 (ID: 15121342) |

### 1-2. 왜 지금이 기회인가 — 경영혁신·창업학적 프레임워크

**블루오션 전략(Kim & Mauborgne, 2005)** 관점에서 현재 국내 건물 에너지 관리 시장을 분석하면, 대형 건물(BEMC 대상, 연면적 3,000 m² 이상)은 이미 의무 에너지 관리가 시행되고 있어 레드오션이다. 반면 **중소형 빌딩(연면적 500~3,000 m²)은 의무 규제 밖이면서도 전체 건물부문 에너지 소비의 45% 이상**을 차지하는 비어 있는 시장이다[추정].

Clayton Christensen의 **파괴적 혁신** 이론으로도 설명된다. 기존 BEMS(Building Energy Management System)는 도입 비용이 건물당 수천만 원에 달해 중소형 건물주가 접근하기 어렵다. 빌딩에코는 공공 API 기반의 **저비용 SaaS**로 이 계층을 파고든다(로우엔드 파괴, Low-end Disruption).

**JTBD(Jobs-To-Be-Done)** 관점에서 건물주의 핵심 job은 "내 건물이 에너지를 얼마나 낭비하고 있는지, 그리고 어디서 줄여야 돈이 되는지 알고 싶다"이다. 현재 이 job을 충족시키는 저렴하고 즉시 접근 가능한 서비스가 없다.

## 2. 아이디어 구상 및 제안배경 (활용적정성)

### 2-1. 사회문제: 건물부문 에너지·온실가스

국내 최종에너지 소비에서 건물부문(가정+상업+공공)이 차지하는 비중은 **약 19.9%**이며,[^1] 건물부문 온실가스 배출량은 2022년 기준 **5,218만 tCO₂eq**으로 국가 전체의 약 **7.7%**에 달한다.[^2]
한국의 **2050 탄소중립 시나리오**는 건물부문 직접배출을 '0'에 가깝게 줄이는 것을 목표로 하지만,[^3] 현재 진척 속도는 매우 더디다.

| 지표 | 수치 | 출처 |
| :---: | :---: | :---: |
| 건물부문 최종에너지 소비 비중 | 약 19.9% | 에너지공단 에너지통계연보 2023[^1] |
| 건물부문 온실가스 배출량 | 5,218만 tCO₂eq (2022) | 온실가스종합정보센터 2024[^2] |
| 30년 이상 노후건물 비율 | 41% (2023년 기준) | 국토교통부 건축물 현황 통계[^4] |
| 에너지 다소비 건물 지정 기준 | 연간 에너지사용량 2,000 toe 이상 | 에너지이용합리화법 제31조[^5] |
| 에너지 다소비 건물 신고 현황 | 약 3,900개소 (2022) | 한국에너지공단[^6] |

**표 1.** 건물부문 에너지·온실가스 핵심 지표

**노후건물 41%의 문제:** 건축물 노후화는 에너지 손실의 핵심 원인이다. 단열 기준이 강화된 2010년 이전에 지은 건물은 현행 기준 대비 냉난방 에너지 소비가 최대 30~50% 높다는 연구 결과가 있다.[^7] 그러나 중소형 건물주는 자신의 건물이 동종 건물 대비 얼마나 비효율적인지 **비교 기준 자체가 없어** 문제를 인식조차 하지 못한다.

### 2-2. 활용분야·활용빈도·활용범위·중요성

| 항목 | 내용 |
| :--- | :--- |
| **활용분야** | ① 건물주 에너지 절약 의사결정 지원, ② 부동산 중개·임대차 계약 시 에너지 효율 공시, ③ 그린리모델링 사업 수요 발굴 및 사전 타당성 분석, ④ 지자체 건물 에너지 집중관리 구역 지정 |
| **활용빈도** | 건물주: 월 1~2회 대시보드 조회 / 컨설팅 수요 발생 시 집중 사용 (단열·설비 교체 검토 주기 2~5년). 부동산 중개사: 매물 검토 시 1회. 지자체: 분기별 구역 보고서 |
| **활용범위** | 전국 중소형 상업용·업무용 건물 약 22만 동[추정] 1차 타깃. 장기적으로 공동주택 단지, 공공기관, 산업단지 입주 공장으로 확장 |
| **중요성** | 2030 NDC 건물부문 감축 목표(2018년 대비 32.8% 감축)[^3]를 달성하기 위해서는 규제 대상 밖 중소형 건물의 자발적 효율 개선이 필수적임. 빌딩에코는 그 자발적 행동을 유도하는 정보 인프라 역할을 함 |

### 3-4. ④ 창의성·독창성

**빌딩에코의 독창적 접근:**

1. **공공 전력 집계 데이터로 건물 개별 EUI 추정** — KEPCO 계약종별 데이터는 건물 개별 정보가 아닌 집계값이다. 빌딩에코는 건물 유형·연면적·준공연도 매칭 알고리즘으로 개별 건물의 예상 EUI 범위를 추정하고, 사용자가 실제 전기요금 고지서 수치를 입력하면 동종 집계 데이터와 비교하는 **하이브리드 벤치마크** 방법론을 채택한다. 이는 개인정보·영업비밀 없이 공공데이터만으로 개별화된 분석을 가능하게 하는 창의적 설계다.

2. **에너지 절감 = 탄소 절감 = 탄소 크레딧 변환** — 절감된 전력량을 국가 전력 배출계수(2023: 0.4393 tCO₂/MWh)[^8]로 환산해 자발적 탄소시장(VCM)에서 크레딧으로 등록하는 경로를 안내함으로써 건물주에게 **절전의 금전적 인센티브**를 추가 제공한다.

3. **13회 수상작과의 차별화** — 제13회 수상작 중 '재생에너지 기상보정' 아이디어는 발전 예측에 초점을 맞췄으나, 빌딩에코는 **소비(수요) 측 효율화**에 집중한다. '통관도우미'·'자연어분석'은 무역·텍스트 도메인으로 에너지 분야와 겹치지 않는다. 빌딩에코는 건물 단위의 EUI 벤치마크를 공공데이터로 처음 구현하는 점에서 중복이 없다.

**AI 해자 논증 (API 래퍼 금지 원칙 준수)**

빌딩에코의 AI는 범용 LLM API 단순 호출이 아니다. 핵심은 **도메인 특화 분석 파이프라인**이다:

1. **독자 데이터 파이프라인**: KEPCO·에너지공단 공공데이터를 건물 유형·기후권역·준공연도로 정규화한 **EUI 기준선 데이터셋**을 자체 구축한다. 이 데이터셋은 외부 LLM이 갖지 않는 빌딩에코만의 자산이다.
2. **Isolation Forest 이상치 탐지**: 비지도 머신러닝으로 각 건물의 월별 EUI 시계열에서 이상 구간을 탐지. LLM과 무관한 도메인 알고리즘이다.
3. **PBP 계산 엔진**: 건물 물리 모델(열관류율·냉난방도일)을 반영한 결정론적 계산 엔진. "단열재 교체 시 절감 전력 X kWh, 비용 Y만 원, 회수기간 Z년" 형태의 신뢰할 수 있는 수치를 산출한다. 이는 확률적 LLM 출력으로는 대체할 수 없다.
4. **모델 교체가능성 전제**: 기반 AI 알고리즘이 더 좋은 버전으로 교체되더라도, 빌딩에코의 가치는 **누적된 EUI 기준선 데이터셋 + 건물주가 입력한 실제 사용량 이력**에 있다. 알고리즘보다 데이터가 핵심 해자다.
