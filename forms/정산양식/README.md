## 외주용역 정산 제출 순서 (권장)
1. **과업지시서** 사전 검토·승인 (→ [`../biz/3_과업지시서_v1.md`](../biz/3_과업지시서_v1.md))
2. 용역 완료 후 **검수조서** 작성(증빙사진 포함)
3. **거래명세서** + 세금계산서 + 이체 증빙 확보
4. **외주용역 수행결과보고서** 작성(검수조서·거래명세서 첨부)

> 금액은 견적서/과업지시서/결과보고서와 **반드시 일치**시키고, 거래명세서는 `공급가액 + 부가세 = 합계(VAT 포함)` 로 분리 표기한다.
