# 창업동아리 지원금 지출 기준 및 구비서류 (요약)

> 정산양식(거래명세서·검수조서·외주용역 결과보고서) 작성 시 참조하는 **집행 기준 요약**이다.
> 원본 공고 [`../refs/ref-startup-club-notice.pdf`](../refs/ref-startup-club-notice.pdf) 기재 사실이며,
> 실제 사업별 세부 기준은 해당 사업단 관리 지침을 우선한다.

## 1. 지원금 집행 기준 요약

| 구분 | 세부항목 | 주요 집행 기준 및 제한사항 | 비고 |
| :--- | :--- | :--- | :--- |
| **아이템 개발비** | **재료비** | 사업계획서 상의 시제품 제작 시 필요한 재료에 한하여 집행 가능 (대량 구매 시 산출내역 및 향후 계획 사전 제시 필요) | 시제품 제작 이전 홍보비 집행 불가 |
| | **외주용역비** | 외주용역의 적정성 및 과업지시서 사전 검토 후 집행 가능 | |
| **홍보비** | **기타 마케팅비** | 홍보영상 제작, 카탈로그 제작 등 아이템 홍보 및 마케팅 비용 | |

| **지식재산권** | **출원비** | 지식재산권(특허 등) 출원에 소요되는 경비 (성공보수료 등은 집행 불가, 반드시 창업동아리 구성원이 출원인이어야 함) | |

## 2. 주요 금지 및 주의사항
- **회의비**: 주말/공휴일 집행 불가, 밤 10시 이후 집행 불가, 주류 포함 시 전액 불인정.
- **여비**: 단순 이동을 위한 교통비 성격이며, 숙박비나 체제비는 지원되지 않음.
- **시제품 제작 전 홍보**: 시제품이 제작되지 않은 상태에서의 홍보비 지출은 원칙적으로 제한됨.
- **지식재산권**: 성공보수료 등 결과에 따른 인센티브 성격의 비용은 집행 불가.

## 3. 관련 제출 서류 (공통)
*※ 구체적인 증빙 서류는 사업단 관리 지침에 따르나, 통상 다음의 서류가 요구됨*

1. **지원금 사용 신청서**: 집행 전 사전 승인용
2. **견적서 및 비교견적서**: (외주 또는 고가 재료 구매 시)
3. **지출 증빙**: 카드 매출전표(법인/체크카드), 세금계산서 등
4. **결과 보고**: 물품 사진, 회의록(사진 포함), 출장 결과 보고서, 결과물(홍보물·홈페이지 캡처 등)
5. **외주용역 시**: 거래명세서 · 검수조서 · 외주용역 수행결과보고서 (본 디렉터리 양식 사용)
