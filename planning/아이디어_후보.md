# 아이디어 후보 (제14회 산업통상자원부 공공데이터 활용 공모전)

> 검증된 산업부 데이터([`조사_산업부공공데이터.md`](조사_산업부공공데이터.md))에 근거. 각 후보는 ① 탈락요건(산업부 데이터) ② 14회 주제축(지역활력 / AI·기업성장) ③ AI 가점 ④ 13회 수상작과의 차별성을 고려. 채점 매핑은 [`../CLAUDE.md` §2.1-E/F](../CLAUDE.md).
>
> **추천 우선순위: A · C · D** (사업화·차별성·AI 가점·테마 적합도 종합). 아래에서 개발할 1~2개를 골라주면 프로젝트 디렉터리(제안서+조사)로 발전시킨다.

## A. 에너지 가계부 + AI 누진·피크 예측  ⭐추천
- **한 줄**: 전기·가스·난방 요금을 한 곳에서 관리하고, AI가 "이번 달 누진 3단계 진입 OO일 예상", "내일 피크" 를 선제 알림하는 가계 에너지 비서.
- **산업부 데이터**: KEPCO 계약종별/산업분류별 전력사용량(벤치마크), 지역난방공사 열요금정보, 가스공사 용도별 공급량, KPX SMP·발전원별 발전량(실시간 탄소집약도).
- **AI**: 사용 패턴 기반 누진단계 도달일 예측, 요금제·절약 추천 LLM, 발전믹스 기반 탄소발자국 환산.
- **문제/임팩트**: 요금부담·에너지빈곤. 기존 한전ON은 청구·모니터링뿐, 가스·난방 미통합·예측 부재.
- **차별성**: 전기+가스+난방 **통합** + **예측 선제알림**(사후 청구 아님). 13회 수상작과 도메인 겹침 없음.
- **테마/부문**: 지역활력(생활) · 제품·서비스 또는 아이디어.
- **리스크**: 가구별 실사용 원천데이터는 비공개 → 사용자 청구서 입력 + 공개 벤치마크로 보완(설계에 명시 필요).

## B. 동네 안전등급 지도 — 전기·가스 위험지수 + 취약 사각지대 발굴
- **한 줄**: 우리 동네·건물의 전기/가스 위험도를 지도로 보여주고, 점검 사각지대(노후·취약계층·소상공인)를 자동 발굴해 우선순위를 제시.
- **산업부 데이터**: KESCO 다중이용시설 전기안전점검 API(부적합 여부)·안전등급 이력·장소별 감전사고, KGS 가스사고 현황·국내 가스시설 현황.
- **AI**: 위험지수 산정 모델, 점검 우선순위 예측(노후·사고이력·시설밀도 결합).
- **문제/임팩트**: 전기화재 전체의 22.9%(주거 34.6% 편중). **KESCO 자체 AI 예측 플랫폼이 2026 미완성** → 단기 공백 큼.
- **차별성**: 기존은 민원·방문점검 중심, 위험 지도화·사각지대 자동발굴 없음.
- **테마/부문**: 지역활력(안전) · 아이디어 기획.

## C. K-탄소내비 — 중소제조 CBAM 자가진단 + 산단 공급망 매칭  ⭐추천
- **한 줄**: 중소 제조기업이 무료로 제품단위 탄소배출을 간이 산정하고 CBAM 보고서를 자동 생성하며, 같은 산단의 공급망·유휴자원(폐열·부산물)을 매칭.
- **산업부 데이터**: KICOX 입주업체·가동률·생산/수출/고용, KEPCO 산업분류별 전력사용량(전력 배출), KIAT 에너지 관련 성과, KPX 발전원별 발전량(전력 배출계수).
- **AI**: 업종·전력·생산량 기반 탄소배출 추정, CBAM 리포트 자동작성 LLM, 공급망·유휴자원 매칭 추천.
- **문제/임팩트**: **CBAM 2026 시행** 대상 중소 약 1,400개사 중 **78.3% 미인지**, 산단 가동률 69.6% 위기.
- **차별성**: 민간 탄소플랫폼은 유료·대기업 지향 → **무료·중소 지향 + 산단 데이터 결합**. 14회 "AI·기업성장" 테마 정확히 적중.
- **테마/부문**: AI·기업성장 · 제품·서비스 개발.

## D. 수출 온보딩 코파일럿 — 품목 기반 유망시장·FTA·인증 자동안내  ⭐추천
- **한 줄**: 수출초보 중소기업이 제품(HS코드)만 넣으면 "유망 수출국 → FTA 세율 → 필요 인증·비용 → 원산지 충족 조건" 을 AI가 자연어로 안내.
- **산업부 데이터**: KOTRA 해외시장뉴스·국가정보·한국무역현황 상세통계·미국 글로벌 이슈, K-SURE 국가목록.
- **AI**: 품목→유망국 추천, FTA/인증 RAG 챗봇(자연어 질의응답).
- **문제/임팩트**: 수출 1년 생존율 49.2%, FTA 활용률 23%p 격차, 바이어·시장정보 부족.
- **차별성 주의**: 13회 아이디어 대상 "통관·수출 도우미(식품 특화)"와 **도메인 인접** → **유망시장+인증 온보딩**으로 각도를 분명히 차별화(통관 신고 대행이 아님).
- **테마/부문**: AI·기업성장 · 제품·서비스 또는 아이디어.

## E. 리콜 레이더 — 바코드 스캔 위해제품 즉시확인 + 개인화 리콜 푸시
- **한 줄**: 마트·중고거래·직구에서 바코드/이미지를 스캔하면 리콜·KC미인증·위해이력을 즉시 확인하고, 내 구매이력과 매칭해 리콜을 푸시.
- **산업부 데이터**: 국가기술표준원(KATS) 제품 안전인증·리콜 정보 API(실시간·무료)·국외리콜정보.
- **AI**: 바코드/이미지 인식, 구매이력-리콜 매칭 알림, 직구 제품 위해 사전판정.
- **문제/임팩트**: 대기업 리콜 이행률 7%대, 직구 위해물질(납 905배), UPSS가 못 막는 중고·직구 사각지대.
- **차별성**: 기존 제품안전정보센터·소비자24는 pull 검색 → **스캔형·푸시형·직구 사전체크**로 전환.
- **테마/부문**: 지역활력(생활안전) · 제품·서비스.

## F. EV 충전 사각지대 지도 + 대기예측 + 입지 추천
- **한 줄**: 농어촌·기축아파트 등 충전 사각지대를 지도화하고, 실시간 대기시간을 예측하며, 지자체·CPO에 신규 입지를 추천.
- **산업부 데이터**: 에너지공단 급속충전기 지역별 충전정보(실사용량), KEPCO 전기차 시간대별 충전부하·지역별 충전소 현황·운영정보. (환경공단 실시간 상태는 **보조**.)
- **AI**: 대기시간 예측, 수요-공급 갭 기반 입지 최적화.
- **문제/임팩트**: 누적 고장률 17.6%, 수도권 45% 집중, 1기당 494대.
- **차별성**: 기존 앱은 위치·가용성 중심 → **사각지대·대기예측·B2G 입지추천**.
- **테마/부문**: 지역활력(지역격차) · 제품·서비스.
- **주의**: 충전 실시간 상태의 풍부한 데이터가 환경부(환경공단) 소관이라, **산업부 데이터(에너지공단·KEPCO)를 핵심 축**으로 설계해야 탈락요건 충족.
