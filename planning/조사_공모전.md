# 조사 — 제14회 산업통상자원부 공공데이터 활용 아이디어 공모전

> ⚠️ **명칭 정정**: 신청서·구어에서 "산업통상부"로 쓰지만 정식 주최 기관명은 **산업통상자원부(MOTIE)**다. 제출 문서에는 정식명을 쓴다.

## 공모전 개요 (제14회, 2026)
| 항목 | 값 | 출처 |
|:---|:---|:---|
| 주최/주관 | 산업통상자원부 + 산하 공공기관 공동 | [1] |
| 접수 | 2026-03-30 ~ 2026-07-06 (기획서) | [2] |
| 총상금 | 5,700만원 (대상: 아이디어 800만 / 개발 1,800만 + 장관상) | [2] |
| 자격 | 누구나, 개인 또는 **최대 5인 팀** | [1][2] |
| 후속지원 | 청년인턴 채용 우대·수출 컨설팅·**GPU 3개월(10팀)** | [1][2] |

> 직전 13회(2025): 3개 부문·총상금 1억·21팀 수상 → 14회는 **2개 부문·5,700만원**으로 개편.

## 주제·분야 (14회)
- **주제: "공공데이터로 견인하는 지역활력, AI 데이터로 도약하는 기업 성장"** [1][2]
  - 두 정책 축: ① **지역활력**(지역 문제·격차 해소) ② **AI·기업성장**(AI 결합 기업 동력)
- **부문 2개**: ① 아이디어 기획 ② 제품·서비스 개발
- 특정 산업 테마로 못박지 않고 **산업·에너지·무역 데이터 전반 + AI 결합** 포괄.

## 평가·가산점 (2단계: 서류 → 발표)
- 1차 서류: 아이템 중복·독창성·개발/활용 가능성
- 2차 발표: 기획 완성도·사업화 가능성·기술 우수성·분석방법 적절성
- **가산점: AI(인공지능)·빅데이터분석 등 미래기술 활용작에 가점** [3][4] — 14회는 AI 결합을 전면에 둠
- 상세 채점 배점·기준은 [`../CLAUDE.md` §2.1-E](../CLAUDE.md) 의 역산 채점표 참조(사용자 제공 평가요소 기준).

## 데이터 활용 의무 (탈락요건)
- 13회 공식 문구: **"산업부 및 산업부 산하 공공기관 데이터를 활용"** 해야 참여 가능 [4]
- 14회: **"산업통상부와 산하 공공기관 데이터를 활용한 프로젝트라면 누구나 참여"** [1]
- → 산업부 **본부 + 산하기관(한전·KOTRA·전력거래소 등) 데이터 활용이 필수 전제**. 타부처/민간 데이터는 보조 결합만 가능.

## 역대 수상작 (13회 대상) — 차별화 기준선
| 부문 | 작품 | 내용 |
|:---|:---|:---|
| 아이디어 기획 | 나만의 통관·수출 도우미 (숭실대) | 식품 통관·수출 지원 |
| 제품·서비스 | shannon — 자연어로 데이터 질의 (중앙대) | LLM 기반 데이터분석 대중화 |
| 데이터 분석 | MLP-XGB 기상예측 오차보정 (SSAFY) | 재생에너지·전력안정성 |

> 시사점: **산업부 데이터 + AI/LLM 결합**이 수상 코드. 무역·통관 / 데이터분석 대중화 / 재생에너지 영역은 이미 수상 → **신규 아이디어는 이와 겹치지 않게** 차별화한다.
