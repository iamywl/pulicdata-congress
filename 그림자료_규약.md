# 그림자료 규약 — 논문형 순수 흑백 통일 (사용자 지시, 2026-06-18)

> **문서(`biz/`)에 들어가는 모든 그림자료(다이어그램·도표·차트)를 학술 논문 figure 처럼 순수 흑백으로 통일**하기 위한 별도 규약. `CLAUDE.md` §2.0 의 단일 출처 상세판이다.
> 레퍼런스: 학술 논문(예: Firecracker/Lambda)의 그림 — 흰 배경·검은 얇은 테두리·검은 serif 글자·채움 없음.

## 1. 원칙 (검정·흰색 2색만)
- **흰 배경(`#ffffff`) + 검은 선·글자·테두리(`#000000`)만.** 회색 채움·음영·컬러 **전부 금지**.
- 도형은 **흰 바탕에 검은 윤곽선**(line art). serif 글꼴(Georgia/Times 계열).
- 모든 figure 에 **번호·캡션**(`**그림 N.** …` / `**표 N.** …`), 본문은 "그림 N/표 N"으로 인용.
- ⚠️ **적용 범위는 문서(`biz/`) 그림자료뿐.** 데모 앱(`projects/`) UI 는 예외 — `design-guide/`(컬러)를 따른다.

## 4. 일괄 변환 + 렌더 눈검수 (의무)
1. 변환: 각 ```mermaid 블록 첫 줄에 §2 init 삽입(기존 init 교체), 컬러 `classDef/class/style/linkStyle`·인라인 `:::` 제거. mindmap 은 flowchart 로 수기 변환.
2. **렌더 검수**: `npx -y @mermaid-js/mermaid-cli mmdc -i fig.mmd -o fig.png -b white` 로 PNG 생성 → **직접 눈으로 열어** 회색·컬러가 없는 **순수 흑백**인지 확인(앱 캡처의 반응형 눈검수와 동일 원칙). 색이 남으면 수정·재렌더 반복.
3. 끝나면 mermaid-cli 설치물(node_modules) 삭제.
